# docs/ja 気づきメモ

書き方（摩擦ゼロ優先。深く考えず1行足すだけ）:

```
- <どこ>｜<気づいたこと>   （任意でメモ: なぜ変 / たぶん正しい形）
```

- **<どこ>** はうろ覚えでOK。あとで探せる手がかりを何か1つ:
  ページパス / URL / grepする語 / スクショ名 / ゲーム状況（どのルート・イベント中か）
- **分類しない・案は無くていい**。「なんか変」だけでも記録する価値あり。
- 迷ったら短く。整形は後（intakeに移すとき）にやる。
