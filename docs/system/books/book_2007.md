# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2007"></BookItemIcon>

峨嵋還是劍派時，遺傳下來的古老鍛身法，化繁為簡，大開大闊，剛猛之處，卻又峰迴路轉，留有餘地。
<br><br>
八路斬法看似平凡，其實，練不出高深武功，但於強健體魄、拉身筋骨助益匪淺。
