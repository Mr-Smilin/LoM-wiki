# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2006"></BookItemIcon>

這套劍法起草自儒家孔夫子手筆，與騎射之術一併傳授弟子的劍術，及後孔夫子周遊列國，網羅七國劍法，不斷改良精進，身後更經數朝更迭，精益求精，如應「天行健，君子以自強不息」的卦象，剛毅圖強，永不停息。
<br><br>
行己也恭，無所爭，無適，無莫，臨大節而不可奪，是謂謙仲平和之劍，君子之劍。
