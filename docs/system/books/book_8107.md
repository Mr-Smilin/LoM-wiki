# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8107" :style="'float: right;'" />

全名為《黃帝八十一難經》，乃著名醫學著作，旨為闡發《黃帝內經》的疑難與要旨。
<br><br>
傳為名醫扁鵲所著，強調「五邪相干」、「十變」理論，解釋五臟生剋關係，
<br><br>
論脈學、經絡、臟腑乃至針法的論述，對後世醫學影響深遠。
<br clear="all" />
