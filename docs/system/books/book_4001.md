# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4001"></BookItemIcon>

嵩山派名揚天下的硬功之一，由踢樁功為根基，習練者雙腿繫沙袋，輪流踢踹木樁百次為一趟，初習者分早午晚日踢三趟，隨著功力日深，一日踢上十趟、二十趟都是日常。
<br><br>
及至百日，除卻沙袋，腳腿輕靈而勢猛如昔，根基已成，再練技巧，彈、蹬、踹、砸、擺、點、撩、掃，八字腿訣，無一不是腳到擒來。
