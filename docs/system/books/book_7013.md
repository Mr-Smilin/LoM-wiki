## 備註

- 此書為龍淵七絕武功節錄的總輯，[[龍淵]]的個人心得薈萃
- 旁人得之，沒有七部武功的根基也學不成，但[[趙活]]有忘形篇
- 龍家七絕是斬落星辰綱、割山掌、射日腿、小霸王槍、離奇點穴繩、[[玄玄如意指]]、鴻蒙造化功
- 斬落星辰岡是一種劍法，跟全真北斗七星陣有關 [^1]
- 小霸王槍則是戰陣中領悟的槍法，跟尋常武功截然不同，走的乃是大開大闔，沙場上對群戰的招數 [^1]
