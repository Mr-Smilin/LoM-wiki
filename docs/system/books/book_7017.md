## 獲得方式

- 青城留學，十一月上與段智秀對質，煉藥總花費點數>60，選擇【那封信】後，再選擇【幕後主使】獲得，注意順序不可顛倒。

## 備註

- 大理段氏天龍訣，枯榮神功。這種掌力揉和陰陽兩屬，離時相生相引，合時相剋相斥，有若雙龍。
- 枯榮神功第三重境界 － 滅・諦・掌！
- 非常麻煩的武功，請搭配丹藥調整陰陽。
- 列表內力、體力沒有填錯，確實是一下升一下降。
