1. 青城留學，二年十二月上葉雲舟歸來時，葉雲裳≥30且技能《醫術》≥3，聽說了寂滅三摩地
2. 葉家兄妹去留轉盤【葉雲裳抵抗】
3. 心上人是葉雲裳
