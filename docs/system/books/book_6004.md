# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6004"></BookItemIcon>

曾經名動一時的怪盜鐵面人所獨創的爪功。
<br><br>
不循正理，而是劍走偏鋒，潛於昏冥，化身為影，遁入黑夜，施以致命一擊。
<br><br>
鐵面人受命於龐太師，意欲襲殺包拯，反心折於包公鐵面無私的氣魄，不願以私戕害<br>
青天，亦不敢背信，遂自捨優勢，暗中行出，向南俠展昭請戰，敗得光明正大，自斷<br>
十指，包拯遂赦免死罪。
<br><br>
鐵面人歸隱於崆峒，請人鑄造鐵爪義手，寫就此書。
<br clear="all" />
