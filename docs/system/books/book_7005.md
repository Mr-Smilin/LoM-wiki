# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7005"></BookItemIcon>

道門蒼龍吟，是以九霄龍吟為譬，形容得道之人飛升清嘯，有如龍吟破空般清亮。
<br><br>
此功須由丹田內氣外發，與人交手之際，發功呼嘯，聲如雷霆疾瀉千里，使人心魄<br>
為之所奪，甚至還能造成內傷。
