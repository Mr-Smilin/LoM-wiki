# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7003"></BookItemIcon>

傳自西域密宗的神祕禪功，習練方法與中原武術截然不同，不單姿勢古怪，遣詞亦十分晦澀，最終二章譯得牛頭不對馬嘴，難以閱讀。
<br><br>
練此禪功講究身體力行，需點燃炭木，鋪以尖銳碎石，再於石上練功，練功者必須潛運內功，將熱力排出體外，否則必將引火燒身。
<br><br>
據聞修練至最高境界可以打坐浮空、手腳關節脫臼伸長打人、口吐烈火。
<br><br>
但不知最後二章的內容，這個境界是無論如何難以得見的。
