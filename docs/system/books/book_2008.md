# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2008" :style="'float: right;'" />

青城秘笈，根據《奇門遁甲》中八門方位、星象、地形所制定的單人陣法。
<br><br>
將此陣法分拆化簡，融會入刀招，即成為八門鎖金刀，精妙無窮。
<br clear="all" />
