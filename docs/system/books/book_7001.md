# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7001"></BookItemIcon>

丐幫人人都會唱的幫歌蓮花落，其實是講究求生技巧的一部內功歌訣。
<br><br>
每夜收工以後，由輩分較老的乞丐帶著小輩齊唱，邊唱邊以筷敲碗，以此掌握節拍，配合歌曲換氣，吐納習慣自然養成。
只不過歌詞往往粗鄙不堪，有辱斯文，有些對著姑娘家唱還會被報官。
