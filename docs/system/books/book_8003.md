# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8003" :style="'float: right;'" />

作者葉雲裳，字跡很娟秀，筆鋒有些稚趣，就像她的人一般。
<br><br>
這書寫了她的雪仗技巧，煞有其事分了《正攻》、《奇襲》兩篇，此書中的招數，多數是她想像出來的。
<br><br>
包含了在雪球裡藏石頭、毛蟲或者雪球塞褲管等等無所不用其極的殘忍伎倆。
