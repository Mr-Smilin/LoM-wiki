# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4005"></BookItemIcon>

一本古老的指譜，由梵文寫成，深藏於藏經閣之下，若非地震，恐怕永世不見天日。
<br><br>
那由他是為佛教數字概念，意為「多不勝數」。
<br><br>
劫則是天竺的時間通稱，歷經成、住、壞、空之後，即為一大劫，故行此功須參悟禪<br>
機，不生不滅，以色身歷劫無盡，方始證悟大道。
