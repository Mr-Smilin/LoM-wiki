# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8104" />

唐門祖傳醫書，當世唯一以毒用藥的醫書。
<br><br>
揉合苗疆、佛家學說，觀一碗水有八萬四千蟲，蟲即為毒，亦分五行六合，對應五臟六腑，多則害，少則病，盡則虛，錯則狂。
<br><br>
人非不害於五毒，實乃毒性未發而已，唯得用藥養之，使其順心不逆，軀毒為己用，則百病難侵。
