## 流程

青城留學期間，進行活動累積功勳點數，計點如下：
+ 上清宮修真 +1
+ 上清宮道士 +1<br>
(誤認地獄道時+0，拖五回合鬼卒助陣+1)
+ 常風觀醫館－青城弟子求診 +2
+ 青城山鎮散步－吟嘯、似有所悟、輸給路人俠 -1

二年九月下鄒博歸來後，拜訪上清宮，若功績點數≥6，觸發嘉勉事件，選擇【拜入青城】。
