## 條件與流程

<strong>條件：</strong>不留學<br>
**流程：**<br>

1. 第二年二月上旬，直接選擇**📜「退席」**，不去留學。
2. [[唐嬌嬌]]登場比武事件：
    - 第二年二月上旬至第二年四月下旬，在練武場進行團練累積兩次，第二年五月上旬唐門會議前發生[[唐嬌嬌]]登場比武事件。
    - 第二年五月期間，在練武場進行團練第二次後發生[[唐嬌嬌]]登場比武事件。
3. 選擇**📖「我就偏不信，上前領教」**，🗡️ 與[[唐嬌嬌]]決鬥勝利，選擇**📖「好，成親吧」**，進入結局。
