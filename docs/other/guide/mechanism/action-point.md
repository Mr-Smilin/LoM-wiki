## {{ $frontmatter.title }}

行動點代表可以互動的次數  
初始是三點，達成特定成就可以增加  
每次互動會消耗行動點，當行動點使用完、會進入到下一個時間段

<br>

每月有**上旬**、**中旬**、**下旬**三個時段  
有些事件**在特定時段沒有觸發就會錯過**  
玩家需要精打細算，在有限的行動中提升武功，觸發奇遇，與各種緣分產生牽絆。
