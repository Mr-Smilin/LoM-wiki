# {{ $frontmatter.title }}

[//]: # (<BookItemIcon :size="`medium`" :needLink="false" :no="7010"></BookItemIcon> : 缺圖)

現今已不存在的雪山派心法，修練方法非常特殊。
<br><br>
需吸足真氣，赤身裸體、頭下腳上插進雪中，只將腳掌露在外邊，再由旁人以艾炙足底湧泉穴，使修練者運使心法借艾炙的熱力溫經通絡，調適體溫、抵禦冰雪。
<br><br>
沒有雪水的話，就近找條清涼的溪流，也可以替代。
