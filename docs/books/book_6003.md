# {{ $frontmatter.title }}

崆峒派．飛天門的傳世武學。
<br><br>
身法輕靈，拳快步鬆，最是敏捷多變。時而漫步閒遊，時而振翅急襲，叫人捉摸不透，防不勝防。
<br><br>
只是這路拳法過於複雜，共有三十六路、七十二變、百零八種後著，甚難熟練，因此飛天門中反倒鮮少有人深入鑽研。
