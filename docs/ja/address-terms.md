# 呼称表 — 誰が誰を何と呼ぶか

::: info
このページは未完成のスタブです。台詞データから機械的に抽出した呼称を人力でレビュー中で、
まだ全体は精査できていません。誤りに気づいた場合は編集にご協力ください。
:::

儒教文化圏の作品では、誰が誰をどう呼ぶかに身分・年齢・親密度が色濃く反映されます。
本ページは活俠傳の台詞から、主要人物同士の呼称(MOD訳と繁体字原語の対応)をまとめたものです。

- マトリクスの表記: `日本語呼称 (繁体字原語)`。同一表記の場合は括弧省略。一覧では別列に分けています
- 原語は抽出元のデータ由来のため簡体字表記が混在することがあり、その場合は機械的に
  繁体字へ変換しています(変換ミスに気づいた場合は修正にご協力ください)。空欄は原語側が
  呼称を含まない構文に意訳されているなどで対応が取れなかったもの
- 相手・場面によって呼び方が変わる場合は、回数の多い順に複数掲載しています。回数の多いものが
  基本の呼称で、少ないものは特定の場面・関係の進展に応じた使用と見られます
- 出典列は、その呼称の代表的な使用場面に対応するイベントページへのリンクです。対応が
  確認できたものだけ記載しており、空欄はページ未作成または対応未確認です(順次拡充予定)
