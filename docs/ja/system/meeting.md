# 唐門月例会議

::: info ドラフト段階
本ページはデータ採取途中のドラフトです。提案の正確な名称・効果については進行状況で補完予定です。
発動条件のデータは確度が高いため、以下の表をそのまま記載しています。提案の実装効果（採択でゲーム内に何が起きるか）については、今後の検証で記入します。
:::

## 概要

唐門月例会議は、月例で開催される唐門の会議に関するシステムです。特定の条件を満たすと、会議の議題として提案ができます。

- 一度提案すると、**再提案不可**になるものが多い
- **ステータス名の対応**: 門派規模／名声／総鍛造／総煉丹／医術／道徳／想い人
- **「3年8月上旬を除く」ゲートの正体**: 3年8月上旬は「四師兄道別」([唐惟元](/ja/people/characters/brother4)との別れ)という一回限りの
  メインストーリーイベントの発生時期。この月だけ通常の
  提案が全てロックされ、代わりに「唐惟元の遠行に関する専用議題」に差し替わる、という排他構造。
  名声・門派規模等の条件はこの特別月以外ならいつでも判定される

## 提案採択メカニズム

- 各提案は採択すると1度限りの施行に固定される
- 提案によっては貢献度・資産の消費量に応じてさらに細かい分岐を持つケースもある

## 名声(名聲)の稼ぎ方 (待検証：外部攻略情報からの参考情報)

唐門ルートでは名声の安定した入手手段が乏しいとされる。外部攻略サイトの情報によると:

- 外堡を買い戻した後、管家として働くと低確率で名声が増える(ゲーム内データ未裏取り)
- 薪拾い/水汲みの「路人俠」イベントは名声+1(勝利時)がゲーム内データで確認済み
  ([唐門の訓練](/ja/system/training/index)「路人俠」の項目、既存記載)

上記1点目(外堡管家)は外部攻略情報のみでゲーム内データの裏取りはできていない。

## 会議がロックされる月

時期条件が付く提案は「3年8月上旬」「3年12月上旬」の2つのみで、他の月への除外は存在しない。

- **3年8月上旬**: 章6-1「四師兄道別」。この月だけ通常提案が
  全ロックされ、四師兄道別専用議題に差し替わる排他構造(上記参照)。
- 1年10月の「壽宴準備」(南宮家宴)は別の仕組みで
  実装されており、月例会議のロジックには関与しない。外部勢力絡みのイベントは月例会議とは
  独立したシステムとして扱われている。
