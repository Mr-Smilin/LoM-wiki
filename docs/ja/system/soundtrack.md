- Steam：[Vol.1](https://store.steampowered.com/app/2983230/)、[Vol.2](https://store.steampowered.com/app/3968660/)
- Youtube Music：[リンク](https://www.youtube.com/channel/UCJ_80xb_-ujC70eD9PjOsIA)
