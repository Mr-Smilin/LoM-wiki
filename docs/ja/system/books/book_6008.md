# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6008"></BookItemIcon>

あなたの唐門大師兄がその登徒子めいた行跡の中で自ら創造した武学で、登雲梯続集に収録されています。
<br><br>
美女に会うため、雲に登りて楼台に登ることができます。その中で踏月式はこの軽功から派生した攻撃招です。「月」は光輝を指し、暗に自分に加えられた兵刃すべてを示しています。
<br><br>
故に踏月式の真義は、他人の兵刃に踏まれ、居高臨下に相手の顔を踏みつけるのです。

## 習得方法

-   2年目崆峒派留学で[夏侯蘭](/ja/people/characters/girl5)に師事時、奪魄森林で修行。師父の不機嫌が累積2回後に[唐布衣](/ja/people/characters/brother1)についての話が出て、再び森に入り、話題で大師兄を選んで習得。

## 備考

-   [夏侯蘭](/ja/people/characters/girl5)が著した、[唐布衣](/ja/people/characters/brother1)の『登雲踏』と雪山派の『梅梢望月歩』を揉合わせたもの。
