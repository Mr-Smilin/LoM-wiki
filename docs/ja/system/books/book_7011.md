# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="`7011`"></BookItemIcon>

> 先入観に束縛されず、他人の長所を受け入れ、謙虚に自分を磨き、平心静気で内に省みる。

錦香宮の主が心血を注いで創制し、南宮心法から派生し、人の心意を穏やかに定め、悲哀で身を傷つけるのを防ぐことを目指しています。
<br><br>
玄燭は月を意味するため、この功はまた望月心経と呼ばれます。
<br><br>
人は悲歓離合がありますが、月は陰晴圓缺があり、この事は古来より全からず、しかし天が各々に離れていても、照らすのは同じ一片の月光です。
<br><br>
慟動する時は常にこの功を行い、心神を収斂し、悲しみを力に変え、きちんと生きていきます。

- 錦香宮不伝の秘
