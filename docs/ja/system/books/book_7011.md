# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="`7011`"></BookItemIcon>

> 先入観に束縛されず、他人の長所を受け入れ、謙虚に自分を磨き、平心静気で内に省みる。

錦香宮の主が心血を注いで創制し、南宮心法から派生した功法で、人の心意を安定させ、悲しみで身を傷つけることを防ぐことを目指しています。
<br><br>
玄燭は月を意味するため、この功はまた望月心経と呼ばれます。
<br><br>
人は悲喜離合があり、月も陰晴円缺がある。この事ぞ古来より全からず、しかれど各々に離れていても、照らすは同じ一筋の月光なり。
<br><br>
心が痛む時は常にこの功を行い、心神を落ち着かせ、悲しみを力に変え、良く生きていきなさい。

## 備考

- 錦香宮不伝の秘
- この功法は本質が陰寒で、心の乱れと火気を調伏できる
- 温夫人がこの玄功を創制したのは、門派の弟子たちが心を落ち着かせ、激情に迷わされず、冷静に判断できるようにするためである。<br>心に障碍がなければ、過去を振り返るとき、思い出がより鮮明になり、心の決意もより揺るがないものになるのだ。
