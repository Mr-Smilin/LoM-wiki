# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7013"></BookItemIcon>

伝説的な前代武林盟主・龍淵の成名の武功。実は一套の武功ではなく、龍淵がかつて使用した七種類の絶技の総称である。
<br><br>
刀剣：斬落星辰綱；拳掌：割山掌；腿法：射日腿；槍棍：小霸王槍；長索：離奇點穴繩；暗器：如意指；内功：玄玄造化功。
<br><br>
当年龍淵が群雄を号令して、かつての魔教天堂総壇に攻め込み、極楽教主と一人で対戦した。斬星射日、霸王割山を連続で繰り出し、当世の正邪両派の高手を震駭させ、その場の全員が異口同音に思った：「これが今の天下第一だ。」

## 備考

- この書は龍淵七絕武功の節録の総集編であり、[龍淵](/ja/people/characters/special2)の個人的な心得の集大成である。
- 他者がこれを得ても、七部武功の根基がなければ習得できないが、[趙活](/ja/people/characters/player)は忘形篇を持っている。
- 斬落星辰岡は一種の剣法で、全真北斗七星陣と関係がある。 [^1]
- 小霸王槍は戦陣の中で悟った槍法で、普通の武功とは全く異なり、大きく開いて大きく閉じる、戦場での群戦への招数を走らせている。 [^1]
- 離奇點穴繩、[玄玄如意指](/ja/system/books/book_4004)は[龍淵](/ja/people/characters/special2)の妻・完顏夢が教えたものである。
- 敘述には玄玄造化功と書かれているが、ゲーム内の対話では鴻蒙造化功である。
- [龍湘](/ja/people/characters/girl8)七絕：斬星射日！霸王割山！玄玄如意... ~~[方少俠](/ja/people/characters/other16)しっかりしろ！殺人だ！白衣の娘さんが殺人を！~~
- 天罡伏魔劍は[龍淵](/ja/people/characters/special2)が七絕武功を融会貫通させた後に悟った最終決戦の奥義である。[^2]

[^1]: [鳥熊 Q&A 集 2024年10月](/ja/other/qna/202410#_2024-10-03)
[^2]: バハムート－[RE:【情報】鳥熊問答集（已更新11/2第二波+私人問）](https://forum.gamer.com.tw/Co.php?bsn=73317&sn=12184&subbsn=1&bPage=0)
