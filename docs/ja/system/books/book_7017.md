## 入手方法

- 青城留学の11月上で段智秀と対面した際、煉薬の総消費ポイント>60、【あの手紙】を選択した後、更に【黒幕】を選択して獲得。順序を変えてはいけません。

## 備考

- 大理段氏の天龍訣、枯榮神功。この掌力は陰陽両属を揉和し、離時は相生相引、合時は相剋相斥し、まるで二匹の龍のようだという。
- 枯榮神功第三重境界 - 滅・諦・掌！
- 非常に面倒な武功なので、丹薬と組み合わせて陰陽を調整してください。
- 表内の内功、体力は記入を間違えていない。実際に上がったり下がったりする。
