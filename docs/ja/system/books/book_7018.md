# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7018" />

青城派の神妙な武功。太極の理を融会し、気を骨に歛め、内に精神を固める。掌心に於いて万物を造化し、一収一放、一陰一陽、一静一動。
<br><br>
来者の勁道が陰であれ陽であれ、いずれも化収することができ、まるで泥牛が海に入るがごとく、消散して無形となる。

## 入手方法

- 青城留学で9月下旬までに、3回修練した後に排擠イベントが発動。[三五斬妖劍](/ja/system/books/book_2009)を獲得。11月下旬に青城を離れる時に地獄道と交戦。下山時に申屠龍≧20で贈られる。
