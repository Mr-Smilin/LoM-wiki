# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7005"></BookItemIcon>

道門の蒼龍吟は、九霄龍吟に例えて、得道者が飛昇して清く鳴く様を龍が空を切き破る清らかな鳴き声に譬えています。
<br><br>
この功は丹田の内気を体外へ発し、人と交手する際に発功して呼嘯し、声は雷霆が千里を疾走するようで、人の心魄を奪い、内傷を与えることもできます。
