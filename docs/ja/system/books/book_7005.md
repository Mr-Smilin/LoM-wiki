# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7005"></BookItemIcon>

道門の蒼龍吟は、九霄龍吟に譬えて、得道者が飛昇して清く鳴く様を龍が空を破く清らかな鳴き声のように形容しています。
<br><br>
この功は丹田の内気を体外に発して、人と交手する際に発動して吼え、声は雷のように千里に轟き渡り、人の心魄を奪い、内傷をも与えることができます。
