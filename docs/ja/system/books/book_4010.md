## 習得方法

- 四書事件
- 3年4月上旬――[唐惟元](/ja/people/characters/brother4)から購入($2500)

## 備考

- この書は修羅道法王の前半生を窺い知ることができます。殺した人は全て二国の交戦する戦場の上にいました。修羅法王が悪人ではなく、むしろ大宋の護国神将ともいえることが分かります。
