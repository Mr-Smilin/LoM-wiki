# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="1000"></BookItemIcon>

唐門弟子人手一本の武学秘笈で、『毒経』と並ぶ二大宝。
<br><br>
師長の伝授なしに心訣・運劇手法がなければ、自学のみでは進展が極めて難しい。
<br><br>
序文の『忘形論』は既に熟読しているはずだ。
