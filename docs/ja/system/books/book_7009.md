# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7009"></BookItemIcon>

飛天門と奪魄門が崆峒の門主選出を巡って競争し、速成を求める余り、火蠍の毒を飲んでやめることで内功を促進し、瓶頸を無理に突破しようとしました。
<br><br>
焚心火印を運用すれば、一時的に心臓の鼓動を加速させ、血流を促進し、残りの寿命を燃やす代わりに肉体能力を全面的に向上させることができます。
<br><br>
しかし代価はそれだけではなく、火毒が体内に蓄積し、最盛期に達するたびに表れます。軽い場合は口の乾きと歯茎からの出血、不眠と易怒、重い場合は火気が脳に衝撃をもたらし、痴呆か急死のいずれかです。
<br><br>
毎年端午の陽気が最盛期の時ほど危険であり、注意を怠ってはいけません。

<BookItemIcon :size="`medium`" :needLink="false" :no="7009"></BookItemIcon>

飛天門と奪魄門が崆峒の顔を争う中、速成を求めて、毒を以て毒を制する方法で、火蠍毒を使って内功の伸長を助け、無理やり瓶頸を突破します。
<br><br>
焚心火印を運用すれば、心搏を一時加速させ、血流を推進し、餘壽を燃焼することを代価に、肉体能力全般を上昇させます。
<br><br>
しかし代価はこれだけではなく、火毒が体内に蓄積され、ピークに達するたびに症状として現れ、軽い場合は口乾、歯茎からの出血、不眠易怒ですが、重い場合は火気が脳に衝き、痴呆か暴斃します。
<br><br>
毎年端午の陽気が最盛の時節は特に危険で、注意が必要です。

## 備考

-   飛天門の秘奥
-   焚心火印はもともと光明正大で、心を平たく静かに保ち、着実に修練すれば、暑気を吸収し、臓腑を鍛錬し、日が経てば自ずと功が成る
-   飛天奪魄両門の争いが激烈なため、飛天門の先輩が速成法門を創出し、火毒を服用して無理やり熱力を催谷熱化させ、臓腑を煎熬します。大関を乗り切って死を免れれば、功が成ります。しかしこの方法で修練する弟子の十に八九は短命で早死にします。
-   [虞小梅](/ja/people/characters/girl3)は天賦が卓越しており、暴斃で死ぬことはなく、[火龍仙君](/ja/people/characters/special204)が運功で火力を疏導してくれたので、最終的に功が大成しました。ただ臓腑は既に傷んでいるため、後患が無窮で、毎回運功する際も心焦のように焚け、劇痛に耐えられず、遠くは寿を減らし、近くは暴斃の危険があるため、全く実戦には使えません。そのため[虞小梅](/ja/people/characters/girl3)は人と交手する際も、焚心火印を催運することはありません。
