# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7009"></BookItemIcon>

飛天門と奪魄門が崆峒の門主を巡って競い、速成を求めるあまり、火蠍の毒で内功を無理やり伸ばし、瓶頸を突破した。
<br><br>
焚心火印を運用すれば、心搏を一時的に加速させ、血流を促進し、余命を燃やす代わりに肉体能力全般を向上させることができる。
<br><br>
しかし代価はそれだけではなく、火毒が体内に蓄積され、ピークに達するたびに症状として表れる。軽い場合は口の乾きと歯茎からの出血、不眠易怒だが、重い場合は火気が脳に衝いて痴呆か暴斃する。
<br><br>
毎年端午の陽気が最盛の時節は特に危険で、注意を怠ってはいけない。

## 入手方法

-   2年目崆峒派留学－飛天門季試三勝報酬（[虞小梅](/ja/people/characters/girl3)好感度≧25）
-   3年目招親－飛天門留学、[金烏上人](/ja/people/characters/special207)に勝利、[顏疆](/ja/people/characters/trainee204)死亡時に贈呈

## 備考

-   飛天門の秘奥
-   焚心火印はもともと光明正大で、心を平たく静かに保ち、着実に修練すれば、暑気を吸収し、臓腑を鍛錬し、日が経てば自ずと功が成る
-   飛天奪魄両門の争いが激烈なため、飛天門の先輩が速成法門を創出し、火毒を服用して無理やり熱力を促進させ、臓腑を傷つけた。大関を乗り切って死を免れれば、功が成る。しかしこの方法で修練する弟子の十に八九は短命で早死にする
-   [虞小梅](/ja/people/characters/girl3)は天賦が卓越しており、暴斃で死ぬことなく、[火龍仙君](/ja/people/characters/special204)が運功で火力を疏導してくれたので、最終的に功が大成した。ただ臓腑は既に傷んでいるため、後患が無窮で、毎回運功する際も心が焦り焚け、劇痛に耐えられず、遠くは寿を縮め、近くは暴斃の危険があるため、全く実戦には使えない。そのため[虞小梅](/ja/people/characters/girl3)は人と交手する際も、焚心火印を運用することはない。
