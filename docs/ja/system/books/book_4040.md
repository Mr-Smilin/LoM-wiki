# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4040"></BookItemIcon>

阿摩という二字がそもそも何の意味なのか、誰も知りません。丐幫の伝功長老でさえ、耕陽読書斎の知識人たちでさえ、知りません。
<br><br>
しかし搪牆の意味は「塗抹」と解釈されており、換言すればこの拳法は、極めて狭い空間こそが最大の効力を発揮する武功です。敵人を壁に押し付けて思い切り叩き、肉漿になるまで壁に塗抹します。
<br><br>
理念は単純ですが、この武功は外功の基礎がなければ成り立たず、寸勁、頓肘、鉄山靠はすべて実打実の実戦功夫です。
