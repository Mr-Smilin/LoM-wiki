# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8010" />

南宮の高祖、武林盟主・南宮智の手書き抄本です。筆力は雄健で、一撇一捺は千軍を横な掃く勢いで、南宮一族の修行と字練習の臨書対象になっています。<br><br>武功を記載していませんが、南宮智が生涯を通じて悟った雑記で、兵法、学問、医術、心の軌跡、現代の流行の衣装、冗談などが含まれています。<br><br>実はこの本はかなり面白いです。

## 入手方法

- 囲城イベント－南宮遠から贈呈（お金を受け取らず対戦、こちらの体力が80%を超える）<br>PS. 道徳がマイナス + 名声が高すぎない
