# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7000"></BookItemIcon>

江湖の経典的なトリック術で、多くの悪党が使用します。
<br><br>
敵が強く、敵わないとき、タイミングを見計らい、自分で功を働かせて体内の経脈を封じ、気血を凝滞させ、脈搏を一時停止させ、あたかも死んだようになります。
<br><br>
敵が去った後、掌に予め隠した石灰が発熱して自分を目覚めさせます。
<br><br>
ただこの内功の使用は相当なリスクがあり、万が一、目覚めのメカニズムが失敗すれば、目覚めた後も脳酸素欠乏で永遠に痴呆になります。また万が一凶暴な人物を怒らせてしまうと、倒れたのを見て躊躇することなく短刀を突き刺す恐れがあり、仮死が本死になります。
