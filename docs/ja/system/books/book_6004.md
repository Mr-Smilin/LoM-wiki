# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6004"></BookItemIcon>

かつて名を馳せた怪盗鉄面人が自らが創造した爪功です。
<br><br>
正道ではなく、剣走偏鋒、暗冥に潜み、影に化し、暗夜に遁じ、致命的な一撃を施します。
<br><br>
鉄面人は龐太師の命を受け、包拯を襲殺しようとしましたが、包公の鉄面無私の気迫に折服し、私心で青天を害することを望まず、また背信を恐れて、優位を自ら捨て、暗中に行動し、南俠展昭に決戦を申し込み、光明正大に敗北し、十指を自ら断ち、包拯は死罪を赦免しました。
<br><br>
鉄面人は崆峒に隠居し、人に鉄爪の義手を鋳造させ、この書を著しました。
