# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6004"></BookItemIcon>

かつて名を馳せた怪盗・鉄面人が独自に創造した爪功である。
<br><br>
正道に従わず、奇想天外な手段を用いて暗闇に身を潜め、影と化して夜に身を隠し、致命の一撃を放つのである。
<br><br>
鉄面人は龐太師の命を受けて包拯の暗殺を図りましたが、包公の鉄面無私の気迫に心を折られ、私利私欲で正義を傷つけることを望まず、なおかつ背信することもできず、自ら優位を手放して暗中に身を隠し、南俠展昭に決戦を挑み、堂々と敗れて十指を自ら断ちました。包拯はこれを赦して死罪を免じたのである。
<br><br>
鉄面人は崆峒に隠居し、人に鉄爪の義手を鋳造させ、この書を著したのである。
