# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4003"></BookItemIcon>

峨嵋派の白猿祖師はある日、暇をみつけて山間に住む猿の友人を訪ねて遊ぼうとしていたところ、思いがけずこの人生において永遠に忘れられない奇想天外な光景に遭遇しました。

山猿が小白兎を手で掴み、蛇が猿猴の身体に纏わりつき、老鷲は天より降りて蛇を撲ち、小白兎は飛び脱けて老鷲の眼を踏みにじりました。

この光景はわずか一瞬のうちに過ぎ去りましたが、四獣は皆、大きな驚きを受けて四散逃亡し、ただ白猿祖師だけは呆然として暫時茫然自失し、家に帰ったあともご飯が喉を通らず、ついに三日目に悟りを開き、朝食のときに弟子たち全員の前でこの蛇兎鷹猿搏を演じました。

弟子たちは皆、それを実演して検証し、資質の高低に応じて理解の深さに大小の差が生まれました。しかし、ただ一人だけが兎型拳を悟ることができました。

## 習得方法

- 2年目の崆峒派留学期間に[夏侯蘭](/ja/people/characters/girl5)に師事し、まず『[つよつよ蹴り](/ja/system/books/book_4002)』を習得してから、3年目の招親イベント後に後山で修練して参拝する。
