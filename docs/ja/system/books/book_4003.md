# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4003"></BookItemIcon>

峨嵋派の白猿祖師はある日の閑時に、彼の山間の猿の友達を訪ねて遊戯しようとしましたが、思いがけず彼を永生涯忘れられない奇抜な光景に撞着しました。

山猿が小白兎を手で掴み、蛇が猿猴の身体に纏わりつき、老鷲は天より降りて蛇を撲ち、小白兎は飛び脱けて老鷲の眼を踏みにじりました。

この光景は瞬間即逝ですが、四獣は共に甚大な驚嚇を受け、四散逃亡し、唯独白猿祖師は怔然たること半晌で、屋に返ってからは茶飯が進まず、終に第三日に融会貫通し、早飯の時に拍案自ら弟子の面前で這套蛇兎鷹猿博を演示しました。

衆弟子は各々印証し、資質の高低に従い領悟に多寡の別がありました。ただ一人が兎型拳を参悟しました。
