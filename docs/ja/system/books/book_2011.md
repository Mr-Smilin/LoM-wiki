# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2011" />

静江九県小盟主・方震天の成名絶技で、本は奇門の武学で、鉄槌と釘を合用して施展するもの。
<br><br>
機縁巧合で方震天が手に入れたが、これは自分が苦心創制した秘伝武学だと称し、人に看破されないよう、本来の槌法を刀法に改めて、釘法は省いてしまったため、平かに意義不明な怪招が増え、隙も多い。

## 入手方法

- 青城留学9月下旬前に、修練を3回実施して排擠イベント発生。ルーレット【申屠龍抵抗】で點蒼棒法を獲得。9月上旬～2年10月中旬に雲裳が秘笈を送ってくる。ルーレット【盤古斧】で獲得。
