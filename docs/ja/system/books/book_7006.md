# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7006"></BookItemIcon>

効果が強力で流伝も広いですが、誰も修練したくない気功です。これは童子の功だからです。痛ましいことです。
<br><br>
聞くところによると、結婚すれば、苦労して修練した功力は散ってしまい、誰も孤独に終わりたくないので、この武功を修練しようとしません。しかしこれは世間の誤解です。
<br><br>
いわゆる童子功とは、先天の気を保留して後天を養うことを指しており、したがって幼少期から修練するのが最も効果的という意味です。母親の胎内から誕生した時から修練すれば、十二年で大成します。志学の年から修練すれば、筋骨は既に成型され、先天の気は逸散し、効果は半減します。而立の年で再び学べば、もう無意味です。結婚の有無とは全く関係ありません。

護体気罩－内功の50%に基づいて防禦を上昇させ、突く人と絶招からのダメージの75%～100%を反射します。反射効果が発生するまで継続します。

## 入手方法

- 千面人イベント－外門弟子オプション選択、雑談は試験待ち（心相が憂鬱でないことに注意）
- 食事に招待しても同様に入手できる可能性があります。龍湘について話さないか、龍湘が呆けていることを表す場合に必要です。
