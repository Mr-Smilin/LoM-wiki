# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2301"></BookItemIcon>

蒼松劍客・葉雲舟が劍を習って半生、得たものすべて。風雨山で恩師と訣別するその刻に始めて悟りに達す。
<br><br>
剣を繰り出せば迅雷のごとく、点蒼のあらゆる快剣をも凌ぐ — それが唐門の剣法である。

## 入手方法

- 葉家兄妹が武林大會に出席し、[葉雲舟](/ja/people/characters/special3)が[唐門](/ja/people/factions/tang-men)代表として出戦。3年12月に武林盟の檄文を受け取り、葉雲舟の意見を求めて獲得。

## 備考

- 本来は「沈舟劍譜」と呼ばれていて、釜を破り船を沈める（退路を絶つメタファー）、勝利を求めるために手段を選ばず自分の後路を断つ意味だった。だが[趙活](/ja/people/characters/player)は縁起が悪いと感じ、[葉雲舟](/ja/people/characters/special3)に「お前は思う存分勝利を求めればいい。背後は俺が守る」と伝えた。そこで「同舟劍譜」と改名され、風雨同舟・同舟共済の意味を持つようになった。
