# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7003"></BookItemIcon>

西域密宗から伝わった神秘的な禅功です。修練方法は中原武術とは全く異なり、姿勢も奇妙で、訳された文言も非常に晦渋で、最後の二章の訳はちんぷんかんぷんに等しく、読むのは困難です。
<br><br>
この禅功の修練では実践を重視し、炭木に火をつけ、先端の鋭い小石を敷き詰めた上で修練を行い、修練者は内功を働かせて熱を体外に排出しなければなりません。そうしなければ、確実に火に焼かれることになります。
<br><br>
聞くところによれば、この禅功を最高の境地まで修めることができれば、座禅で宙に浮かぶ、手足の関節を脱臼させて伸ばして人を打つ、口から烈火を吐くことができるようになるとのことです。
<br><br>
しかし、最後の二章の内容が分からないため、この境地に到達することはいかなる方法によってもほぼ不可能でしょう。
