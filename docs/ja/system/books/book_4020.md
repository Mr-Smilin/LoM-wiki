# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4020"></BookItemIcon>

全真派の軽功を兼ねた脚法。逍遙遊は『荘子』一編に出典し、また古琴曲でもあります。
<br><br>
全真派弟子が毎日修行する際に、伝功師父が曲を奏でます。鯤が鵬に化し、翼は垂れた雲のごとく、天地を往き、青空に達し、扶搖万里を御し、風に乗りて冷然として、気適北溟に、磅礴万物、翱翔広和九段奏を終えて、初めて修行を終わります。

## 習得方法

- 1年目弟子房(憂鬱)－大師兄と小師妹でいたずら
- 2年1月上旬－四師兄の物品販売 ($1800)
- 2年崆峒派留学－無色広場で散策、四師兄の物品販売
