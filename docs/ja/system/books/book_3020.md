# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="3020"></BookItemIcon>

苗疆五毒教の用毒法門であり、唐門毒経と相比して各々優位があり、薬理も同じではありません。

異郷の石も、錯誤を為すことができます。

- 医術：殴られている時、最大血量に応じて（1級2%、上限10%）血量を回復させ、追加で毒・痺れの累積値を代謝し、部分的なストーリーに影響します。
- 毒功：突く時に50%の毒・痺れ累積値を付与します。

## 習得方法

- 四書事件
- 3年4月上旬――四師兄から購入($3000)

## 備考

- 初期段階で医術を最も容易に習得できる秘笈です。
