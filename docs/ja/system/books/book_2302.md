# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2302"></BookItemIcon>

点蒼派から万に一つの才を持つ弟子が現れると、格別に剣廬へ三日間進入を許され、点蒼剣聖が剣廬に残した三式の剣意を悟ることができます。三式の剣意は各々奥義を備えており、優劣はなく、一つを得るだけで江湖に独歩できるほどです。しかし十のうち九までは、十日半月滞在してもなお一つも得られません。

葉雲裳は好奇心からこっそり潜入しましたが、ほんの一瞬で連れ出されてしまいました。彼女は武功を修めていませんが、聡明極まりなく、既成概念にとらわれず、その一瞬にわずかながら悟るところがありました。ただし年が幼すぎたため、その悟りはぼんやりとしていたのです。今日の別離で初めて離愁を知り、往事が心頭に湧き上がると、ぱっと悟り、この招式を書き上げました。

ただ一式に過ぎませんが、尽きぬ趣きを備えており、この剣は剣聖第四の極招――「魂遊故里芳盈袖，春啼曉，莫挽留」です。
