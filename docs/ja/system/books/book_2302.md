# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2302"></BookItemIcon>

点蒼派万里挑一の弟子が、格別に剣廬三日に進入でき、点蒼剣聖が剣廬に残した三式の剣意を感得することができます。三招の剣意は各々奥義を備えており、高下なく、一つを得るだけで江湖に独歩できます。しかし十のうち九までは、十日半月滞在しても一無所獲です。

葉雲裳は好奇心から、こっそり潜入して、すぐに連れ出されてしまいました。彼女は武功を知りませんが、極めて聡明で、既成概念にとらわれず、その瞬間にわずかに悟るところがあった。ただし、年が幼く、半ば寝ぼけたままでした。今日の別れの際に、離愁の味わいを知り、往事が心頭に湧き上がり、豁然と悟悟して、この招式を書き留めました。

ただ一式とはいえ、執心無尽であり、この剣はまさに剣聖第四の究極奥義――「魂は故里を遊び、春の音は暁に啼き、留まるなかれ」です。
