# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8007" />

儒家の戦神・孟子が列国を周遊した際に、諸々の才能ある者と舌戦を繰り広げ、言葉の剣で一掃し、唇で槍のように語って、中原を平定した語録です。出典は不明ですが、田舎の伝説によれば、象棋の「砲」は実は孟子の口を指しており、離れた場所から人を殺すことができ、恐ろしいことに方位さえ合えば誰でも殺せるとされています。<br><br>相手がどれほど高い地位にあろうとも、どれほど謙虚な態度で尋ねてきても、孟子の激しい批判を免れることはできません。彼は言い争いのときにすら孫子兵法と内功を使い、人々が言い返すことができないほどになり、叱られ続けて死んでしまうほどです。<br><br>諸子百家を罵倒したが、自分の非難だけは許さず、怒って言い返します。「予豈好辯哉！？予不得已也！！」という言葉は千古の絶唱となりました。

## 入手方法

- 1年4月上旬－四師兄の買物（$1500）

## 備考

- 嘴炮流の必須武功。敵が1度怒ると、即座に3ターンを制圧できます。
- 【核棗釘】との相性が良好です。
