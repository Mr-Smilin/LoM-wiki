# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6001"></BookItemIcon>

嵩山寺から盗み出された拳経は、嵩山武学の根本であり、習得は易しいが極めるは難しいです。
<br><br>
民間に流通している羅漢拳譜は数百年の間に多くの套路を生み出し、それぞれが個性的で、どれも少しずつ異なります。あなたが手にした拳譜が原版であるかどうかも分かりません。でも拾ったからには仏縁があるというもの、さあ修行してください。
