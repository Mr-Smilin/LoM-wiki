# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4006"></BookItemIcon>

峨嵋派の腿法の最たるもので、天に躍り上がり、雲波の詭譎に潜臥して、吞吐無定、鳴かなければやむが、一たび鳴けば人を驚かし、動くことは九天の神雷が雲霄を轟破するがごとく、雷公が天罰を降すがごとく、威力万鈞です。

この腿法には固定套路がなく、招ありて式なし。形意拳の究極の型態であり、象るところは天雷の形であり、即ち雷神腳です。

## 習得方法

以下の条件を同時に符合する必要があります：
- [唐嬌嬌](/ja/people/characters/big_trainee_girl_1)、[唐衫](/ja/people/characters/special811)が唐門に加入し、唐衫が大師兄と共に下山し、武林大会が唐衫を受け入れます。
- [福韞](/ja/people/characters/special206)は好感度が20未満で、唐門に加入していない。さもなくば3年12月下に峨嵋に出使します。
- 西武林ルートで、4年1月上に[唐衫](/ja/people/characters/special811)を派遣して峨嵋派を説得し、帰ってきた後に唐衫と比試して、彼に忘形篇を教えます。
