# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4002"></BookItemIcon>

元々は「風雷神腳」と名付けられ、脚法の使い手として名高い美男俠客「風神」[解無塵](/ja/people/characters/special808)の成名絶技です。残念ながら、餓鬼道法王劉顎に挑戦する際に、「犯劍」に当衆で辱められ、[解無塵](/ja/people/characters/special808)は急怒攻心して嘔血飲敗し、約に依り「風雷神腿」を「很是用力踢」に改名せざるを得ました。

很是用力踢は使い手として極めて凌厲で、速度は迅速で、風雷が引き至るがごとく、山岩を蹴り裂くことができ、所向披靡にて、まことに厳重無比です。

## 習得方法

- 囲城事件――丐幫優勢、客棧撤退(戦術 LV0が必要)で丐幫に遭遇、風神の救場を得て、1000銭支払って習得。
- 2年崆峒派留学――[夏侯蘭](/ja/people/characters/girl5)に拜師、3年招親後に後山で修練して追悼します。
