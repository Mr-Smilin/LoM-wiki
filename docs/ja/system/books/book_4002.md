# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4002"></BookItemIcon>

元々は「風雷神腳」と呼ばれ、脚法の達人として知られた美男俠客「風神」[解無塵](/ja/people/characters/special808)の成名絶技だった。しかし、餓鬼道法王劉顎に挑戦した際、「犯劍」に衆人環視の中で辱められ、[解無塵](/ja/people/characters/special808)は激怒して嘔血し敗北した。約定に従い、「風雷神腳」を「很是用力踢」に改名せざるを得ぬこととなったのである。

很是用力踢は極めて鋭く、速度は迅猛で、風雷が至るがごとき勢いで、山岩をも蹴り裂き、所向披靡の力を発揮する、実に比類なき武技である。

## 習得方法

- 囲城事件――丐幫優勢で、客棧から撤退(戦術 LV0が必要)する際に丐幫に遭遇し、風神に救援されて、1000銀両を支払って習得。
- 2年目に崆峒派へ留学し、[夏侯蘭](/ja/people/characters/girl5)に師事、3年目の招親の後、後山で修行して祭祀を行う。
