# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2006"></BookItemIcon>

この劍法は儒家の孔夫子の手による原稿に由来し、騎射の術と一緒に弟子に伝授された劍術である。その後、孔夫子は諸国を周遊して七国の劍法を集め、不断に改良精進し、身後さらに数朝の更替を経て、一層の精化を遂げたもので、「天行健、君子以自強不息」の卦象に応じ、剛毅で図強、永遠に止まることがない。
<br><br>
己の行いも恭しく、争うことなく、執着なく、執着することなく、大節に臨んで奪われることなし。これが謙仲平和の劍、君子の劍である。
