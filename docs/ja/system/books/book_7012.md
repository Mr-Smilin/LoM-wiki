# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7012"></BookItemIcon>

南宮家が代々受け継ぐ家学の心法で、各自一本を所有し、歌訣は本の内扉に大きく抄録されています。
<br><br>
道家思想を融合させ、万変もなお定かであり、神も怡ぶて気も静かであり、意を玄関に留め、邪気を降伏させ、虚懷若谷に心神は清明なり。
<br><br>
心を守ることを本とし、その後身を護り、家を保ち、国を衛る。これが南宮家学の根基です。

## 入手方法

- 囲城イベント－南宮遠が贈与（金を取らずに切磋、血量が40%～80%）
- 2年10月中旬－四師兄から購入（$2500）

## 備考

- 医術で大師兄を救う予定なら、南宮護心訣を習得することをお勧めします。
- 最高境地は『偕老の誓い』で、陰柔内力と嫡伝内息が水乳交融して一となり、あたかも一縷の香魂のごとく、相護って老いる。
- [葉雲裳](/ja/people/characters/girl2)ルートで技能を発動して救う際、既に南宮護心訣を習得していれば、技能が消滅し、偕老の誓いを習得する。~~南宮遠からの愛~~
- 偕老の誓い：致命的な攻撃を耐えられ、気力100を回復。
