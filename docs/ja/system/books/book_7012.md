# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7012"></BookItemIcon>

南宮家が代々受け継ぐ家学の心法で、人手一つ所有し、歌訣は本の内扉に大きく抄録されています。
<br><br>
道家思想を融合させ、万変も定かであり、神も怡びて気は静かであり、意は玄関を守り、邪気を降伏させ、虚懷若谷で心神は清明です。
<br><br>
心を守ることを本とし、その後身を護り、家を保ち、国を衛る。これが南宮家学の根基です。

## 入手方法

- 囲城イベント－南宮遠が贈与（金を取らずに切磋、血量が40%～80%）
- 2年10月中旬－四師兄から購入（$2500）

## 備考

- 医術で大師兄を救う予定なら、南宮護心訣を習得することをお勧めします。
- 最高境地は『偕老の誓い』で、陰柔内力と嫡伝内息が乳交融の水の様に交わり、合而為一で、まるで一縷の香魂のように、相護って老いる。
- [葉雲裳](/ja/people/characters/girl2)ルート施功で拯救する時、既に南宮護心訣を習得していれば、技能が消滅し、偕老の誓いを習得する。~~南宮遠からの愛~~
- 偕老の誓い：致命的な攻撃を耐えられ、気力100を回復。
