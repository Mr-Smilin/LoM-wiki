# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2303" />

葉雲裳が勝手に点蒼剣譜を改ざんして創作した棒法です。

殺傷力といえば大したことはありませんが、点蒼剣法の足りない部分を補完し、敵の意表をついて、招数の変化だけなら見事といえます。ただし、やや卑劣で、正常な人ではこのようなことはしません。

- 難知如陰：戦術LVに応じて弁舌を上昇させます(1級+1)。

## 習得方法

- 2年青城留学9月下旬前に、3回修練した後に排斥事件を発生させ、スピンボタン【申屠龍の抵抗】
