# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7014"></BookItemIcon>

武林三大世家・上官一脈に伝承する心法で、翻浪刀法の真髄である。
<br><br>
滾湧する激流の中で刀を練う必要があり、激動する波涛の中でも揺るがず立ち、刀を振るって波を斬る境地まで練ることができる。
<br><br>
招式はまさに滄浪の一波が平まず、また一波が起こるがごとく、圧倒的に迫り来る。内功が到るところ、威力さらに増す。相手が招架しても、尚も数歩踉蹌とする。
