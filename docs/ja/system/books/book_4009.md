# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4009" />

青城天下幽にして、山に陰陽三十六峰あり、背は雪嶺に靠り、林木は森々として、四季常青です。

相伝えて唐時に富賈がおり、死を畏れて仙を求め、山下で童子に問いて曰く：「林間に在り」。林間で童子に問いて：「山澗に在り」。山澗で童子に問いて：「巒顛に在り」。一路連問三十六峰にして、また童子に見えて、賈遂に神異を知り、地に拜します。童子笑いて曰く：「孺子可教なり。清修何ぞ苦煩悩を須いん。心謙不躁にして道を問うべし」と。

富賈清修多年にして掌門を接任し、此書を書き下しました。字裡行間には三十六峰を閑遊せし快活心境があり、山に初入の時の怕心は、早已知らぬ間に路上に遺落されてしまった。

## 習得方法

青城留学期間に、活動を進行させて功績点を累積します。計点は以下の通りです：
+ 上清宮の修真 +1
+ 上清宮の道士 +1<br>
(地獄道と誤認する時+0、5ターン鬼卒助陣をドラッグ+1)
+ 青城山鎮医館――青城弟子の求診 +2
+ 常風觀の散歩――吟嘯、似有所悟、路人俠に負ける -1

2年9月下に鄒博が帰ってきた後、上清宮を訪問し、功績点数≥6であれば、嘉勉事件を発生させて習得します。
