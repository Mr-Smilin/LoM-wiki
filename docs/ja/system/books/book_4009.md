# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4009" />

青城は天下に幽深く、陰陽三十六峰という山々に恵まれ、背は雪嶺に接し、林木は鬱蒼として、四季常に青々としている。
<br><br>
昔、唐の時代に富商がいた。死を畏れて仙を求め、山下で童子に問うた：「在りはどこか」。童子は答える「林間に在り」。林間で問えば「山澗に在り」。山澗で問えば「巒顛に在り」。こうして三十六峰すべてを訪ねて回り、再び同じ童子に出会った。富商はここで初めてこれは並の者ではないことを悟り、地に伏して拝礼した。童子は笑って言った。「見どころがある。清く修めるのに、どうして煩悩に苦しむ必要があろうか。心を謙虚に保ち、焦らずに道を問い求めるがよい」と。

富商は清修を重ね、やがて掌門の地位を継ぎ、この書を書き遺した。行間に満ちているのは三十六峰をのんびり遊んだ時の快活な心境であり、初めて山に入った時の恐ろしさは、いつしか路上に取り残されてしまったのである。

## 習得方法

青城留学期間に活動を進めて功績点を蓄積します。加点項目は以下の通りです：
+ 上清宮での修真 +1
+ 上清宮の道士 +1<br>
(地獄道と誤認する時+0、5ターン鬼卒助陣を持たせる+1)
+ 青城山鎮医館－青城弟子の求診 +2
+ 常風觀の散歩－吟嘯・似有所悟・路人俠に負ける -1

2年9月下旬に鄒博が帰ってきた後、上清宮を訪問し、功績点≥6であれば、嘉勉事件が発生して習得できます。
