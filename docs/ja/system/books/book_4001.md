# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4001"></BookItemIcon>

嵩山派が天下に名を馳せた硬功の一つで、樁蹴り功を基本とし、修練者は両腿に砂袋を繋ぎ、交代で木樁を蹴ること百回を一趣とします。初心者は朝昼晩三趣蹴ります。功力が日々深まるにつれ、一日十趣、二十趣蹴ることも日常です。

<br><br>

百日に至れば、砂袋を除いても、両脚は軽捷にして勢いなお猛く、基礎が成就します。その後技巧を修練し、彈、蹬、踹、砸、擺、點、撩、掃の八字腿訣は、いずれも脚が到るところ捉えられます。
