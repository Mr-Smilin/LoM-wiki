# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4005"></BookItemIcon>

古い指譜であり、梵文で書かれており、藏経閣の下に深く隠されていました。地震がなければ、恐らく永遠に日中を見ることはなかったでしょう。

那由他は仏教の数字概念であり、「多不勝数」の意です。

劫は天竺の時間通称で、成、住、壊、空の後に至れば、即ち一大劫となります。故にこの功を行うには禅機を参悟する必要があり、不生不滅にて、色身を以て無尽の劫を歴し、方才大道を証悟します。
