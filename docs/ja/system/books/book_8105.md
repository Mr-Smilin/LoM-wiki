# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8105" />

魏菊が精選した当代の傑出した文人詞集。宋詞100曲を抜粋し、平理は釣り合っており、照辞は鏡のようです。<br><br>収録されたもの必ずしも著名作家の作品ではありませんが、必ず傑作です。<br><br>本書は魏菊の手書きで、字跡は美しく、美女が花簪をするようであり、また鸞が飄び鳳が泊まるような情景で、鬱々として志を得られぬ。文人雅士なら誰もこの冊を見て大喜びするのに、独りあなたというこの江湖人に贈ってしまいました。<br><br>彼女は誓いを厳守しながら、ほんの少しの私心を隠していて、「佚名」の名義で、私心から自分の旧著を収録しています。
