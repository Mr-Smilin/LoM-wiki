# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8105" />

魏菊が精選した当代の傑出した文人詞集で、宋詞百闕を節録したもの。理路は天秤のように整い、詞は鏡のように照らし出す。<br><br>収録の作は必ずしも名家の筆とは限らないが、いずれも傑作である。<br><br>本書は魏菊の手抄で、字跡は娟秀。美女が花簪を挿すようであり、また鸞が舞い鳳が降りるような雅な風情。鬱々として志を果たせず。されど文人雅士ならば誰もがこの冊を見て欣喜若狂するはずなのに、ただあなたというこの江湖人にだけ贈ってしまった。<br><br>彼女は誓約を厳守しながらも、ほんの少しの私心を隠していた。「佚名」の名で、自身の旧著を私心から収録したのである。
