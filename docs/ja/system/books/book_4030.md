# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4030" />

丐幫の人々の間で慣習化した逃げ足の技巧。人数が多いときは非常に壮観で、一哄に散り、四面八方へ奪路して逃げ出し、心が通じ合ったかのように東へ向かう振りをして西へ逃げるなど、敵が一網打尽にするのを難しくします。最終的には必ず何人かが脱出して家に帰り、知らせをもたらすのです。
<br><br>
この逃げ法は歌訣の形で伝承されており、歌は「眼看東邊走西邊、無處可逃上樹顛、三五成群鐵沒命、生死有數莫拖牽」と言います。言葉は俗っぽく浅白ですが、それは敵を騙す知略だけでなく、万が一捕まることに対する心的準備もあらかじめ示しています。
