# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7004"></BookItemIcon>

北宋楊門忠烈の練兵体操です。
<br><br>
体操と言われていますが、実際に戦場での生還率を確実に高めるため、この操の鍛錬強度は朝廷正規軍を遥かに上回り、武林の小門派の功法よりも厳しく、棍操、百八陣操、銅錘十八打が混在しており、この体操は武林の人でも耐える者がほとんどいないほどです。
