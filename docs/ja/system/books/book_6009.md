# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6009"></BookItemIcon>

あなたの恩師[夏侯蘭](/ja/people/characters/girl5)が奪魄、雪山両派の神髄を集めて、ゴミを除き精粋を選び、更に制鶴気功を揉合わせ、やがて創作した神
技です。
<br><br>
狙いは殺傷にあらず、高手に対処するための武学であり、たとえ天下で最高の軽功であっても捕まえられます。
<br><br>
発力はあたかも飛鴻が雪泥を踏むように、泥上に偶然指爪を留め、鴻は飛んでどこへか去るか。軽く淡々と引き抜き、力は三分出して七分隠す、後続技は無形に隠れます。敵の架勢を破壊して、我は素速く回気し、奪命猛襲を施します。
