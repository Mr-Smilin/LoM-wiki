# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4008"></BookItemIcon>

丐幫は蛇の飼育に長けており、丐幫の蛇形拳は単に蛇形を模倣するだけではなく、まさに本物の蛇と共に闘うことです。出拳、出腿は全て蛇との配合が必要であり、啄擊、纏繞擒拿の他に、一記の隠形の殺招が隠れています。招数を使い終わりし時が、すなわち必殺の時です――蛇が不意に袖から出て人を一噛みします！
