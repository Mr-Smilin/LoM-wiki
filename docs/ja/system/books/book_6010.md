# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6010"></BookItemIcon>

極めて狂気的な刀法であり、妻を奪われた恨みを抱いた三流刀客が創始したものです。
<br><br>
不当な罪で13年間獄中にあった間、心の中で何千回も敵の武学を思い描き、脳裡でその仇敵への復讐を何万回も想像しました。
<br><br>
出獄後すぐに、江湖を驚かせた流血事件『碧血狂殺』が勃発しました。
