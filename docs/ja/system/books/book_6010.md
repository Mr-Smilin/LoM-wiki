# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6010"></BookItemIcon>

極度に狂気した刀法であり、妻を奪われた恨みを背負った三流刀客が創始したものです。
<br><br>
不当な罪で獄に下ること13年の間、心の中で何千回も敵の武学を思案し、脳裡でその仇敵を何万回も凌遅しました。
<br><br>
出獄後すぐに、江湖を驚かせた流血事件「碧血狂殺」事件が勃発しました。
