# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="`6012`" />

唐門女俠唐芬が青城の『八仙剣法』の五行学説に基づき、五毒で代替させて創成した独門武学です。
<br><br>
招式に限れば二流程度ですが、隠然として唐門の医毒二術を含み、極めて難解であり、読んだからといって大きな成就が得られるとは限りません。
