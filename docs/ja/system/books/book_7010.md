# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7010"></BookItemIcon>

現在はもう存在しない雪山派の心法です。修練方法が非常に特殊です。
<br><br>
十分な真気を吸収してから、裸身で頭を下にして雪に突き刺し、足裏だけを外に露出させます。側に居る人が足の裏の湧泉穴にもぐさで灸をし、修練者は心法を運用してもぐさの熱力を借り、経絡を温め通し、体温を調整して、氷雪に耐えます。
<br><br>
雪水がない場合は、近くの清涼な小川を見つけても代わりになります。
