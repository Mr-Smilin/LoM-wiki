<BookItemIcon :size="`medium`" :needLink="false" :no="8005" />

江湖の様々な悪知恵を記した警世録です。本来の著者の意図は、無実の民を警告すること、人を騙してはいけない、また人を警戒してもいけないということを知らせることでした。<br><br>しかし、この書が悪人の手に渡ると、すぐに詐欺の宝典に変わってしまい、どのような方法でも使われています。

## 入手方法

- 1年4月上旬－四師兄の買物（$900）
- 千面人事件－外門弟子を選択し、龍湘の話題で「心配させたくない」を選ぶ
- 外出行商－2回お金を盗まれると四師兄が書を送呈

## 備考

- 破廟の貴公子の侠義事件で、1級を修行すると石灰粉を目に浴びた傷を治療できます（医術1または鍛丹ポイント≥30でも可能）
