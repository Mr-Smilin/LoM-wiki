# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8005" />

江湖のあらゆる悪知恵を記した警世録です。著者本来の意図は、無辜な民衆に、人に騙されるな、人の心は測り難い、人を傷つける心を持つな、しかし人を警戒する心は必要だと警告することでした。<br><br>奈何か、この書が悪人の手に渡ると、たちまち詐欺の宝典に変わってしまい、あらゆる手口に使われています。

## 入手方法

- 1年4月上旬－[唐惟元](/ja/people/characters/brother4)の買物（$900）
- 千面人事件－外門弟子を選択し、龍湘の話題で「心配させたくない」を選ぶ
- 外出行商－2回お金を盗まれると唐惟元が書を送呈
