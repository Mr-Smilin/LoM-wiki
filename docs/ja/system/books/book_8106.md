# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8106" />

翰林医官が太宗皇帝の命令で編纂した医書です。<br><br>診察法、処方と薬の使用、五臓の様々な病気、内科、外科、婦人科、小児科、食療、丹薬、外部作用、鍼灸など内容を包括しています。<br><br>原典は100冊に及びますが、あなたが得たのはその綱目のみです。なぜなら二師兄は省略された内容は常識で、あなたが知っておくべきだと考えたからです。
