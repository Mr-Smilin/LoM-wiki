# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8106" />

翰林医官が太宗皇帝の勅令で編纂した医書です。<br><br>診察法、処方と用薬、五臓の各種疾病、内科、外科、婦人科、小児科、食療、丹薬、外治法、鍼灸など、幅広い内容を収めています。<br><br>原典は百冊に及ぶものですが、あなたが得たのはその綱目だけです。[唐錚](/ja/people/characters/brother2)は、省略した部分の内容は常識だと考え、あなたがそれを知っていると判断したためです。
