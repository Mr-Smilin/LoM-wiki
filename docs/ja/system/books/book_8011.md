# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8011" />

火炎山劍閣の各種兵器と鉄器の型録であるだけでなく、秘密にされていた鍛造の秘密も含みます。<br><br>本には異なる金属と合金の特性、処方、火炎山の地熱を利用した焼戻し、焼入れ、焼き戻しなどの工程が記録されています。<br><br>あなたが知っている唐門の鍛法と比べて、より正統で厳格です。
