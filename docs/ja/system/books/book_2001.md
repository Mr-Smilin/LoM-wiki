# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2001"></BookItemIcon>

点蒼派の高深な劍訣で、劍光は流星のように走り、千里一瀉。
<br><br>
飄逸さを求めず、瞬発のみを求める。速劍の二字を窮めて、身劍合一し、意発同時。速さは影さえ追いつけない。
<br><br>
「足踏・曲弓蓄勢・離」、どのような身形でも、この三句の心訣を離れることはできない。その速さは弓矢が弦を離れるようで、瞬間にして敵懷に欺き入り、成功するか自分が倒れるかのいずれかだ。
