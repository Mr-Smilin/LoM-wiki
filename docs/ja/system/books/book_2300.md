# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2300"></BookItemIcon>

点蒼派の劍譜で、軽・柔・快・変の路を走る。
<br><br>
一旦使い始めると、劍意は絶えることなく、行雲流水のように生き生きと優雅に舞う。

- 点蒼快劍：軽功に応じて刀剣が上昇
- 點破雲関：連撃数が2より大きい場合、絶招が【點破雲関】に変わり、現在の連撃数によってダメージに影響。連撃数が8を超える場合、すべての連撃数をクリア。気を12消費。

## 入手方法

- 四書イベント
- 牛を買うイベント－葉家兄妹に金をねだる
- 2年1月上旬－[唐惟元](/ja/people/characters/brother4)から購入
- 2年崆峒派留学－無色の広場で散歩、唐惟元から購入($2500)

## 備考

- 點破雲関は徐如林と相性が良く、連撃数が揃えば連段で最後まで繋げられる
- 点蒼劍法第一式『蒼松迎客』、第七式『白虹貫日』、第十四式『鳳鳴朝陽』、~~六十六式『握手言和』~~
