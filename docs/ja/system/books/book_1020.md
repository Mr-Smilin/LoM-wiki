# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="1020"></BookItemIcon>

天下に聞こえた嵩山指法で、「釈迦拈花、迦葉微笑」の典故から名づけられた。
<br><br>
涅槃妙心は言葉では説くことができず、文字でも記述しがたく、無形無相で不可思議である。この武学の精妙さは、慧根がなければ得ることはできない。
