# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7001"></BookItemIcon>

丐幫の人なら誰もが唱う幫歌蓮花落は、実は求生技巧の内功歌訣です。
<br><br>
毎晩作業を終えた後、輩分の老いた乞食が小輩を率いて斉唱し、箸で碗を叩きながら、拍子を掌握し、歌に合わせて呼吸を変え、吐納習慣が自然に養成されます。ただし歌詞は往々にして粗俗不潔で、品位を損ないます。姑娘家に向かって唱うと報官される場合さえあります。
