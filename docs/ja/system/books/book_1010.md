# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="1010"></BookItemIcon>

飛石幫鎮幫秘笈で、幫主の若き日に高人から授かった秘伝の暗器手法。
<br><br>
修行が深い域に達すると、静かな水面で飛び石で水切りをして、88回も水面を弾ませることができるという。

## 入手方法

- 1年目伙房－大師兄と飛石幫に対戦

## 備考

- 実は[石公遠](/ja/people/characters/special7)が若き日に修得した擲石の心得をまとめたもので、後に[石夫人](/ja/people/characters/special815)が複製し、これを高深武功と称し、門中弟子のみが伝閲できるものだとはばかったが、本当のところ大した武功ではない。
- [唐布衣](/ja/people/characters/brother1)の手にかかると全く別物で、石が螺旋状に跳躍して敵の背後から襲い、一本の気で連珠のように弾き出され、四面八方から石の雨が降るようなもので、実に忘形篇の真髄である。
