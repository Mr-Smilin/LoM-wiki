# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6002"></BookItemIcon>

崆峒派・飛天門の秘笈です。
<br><br>
先掌派の人が『仙鶴迷蹤拳』を簡潔化して弟子に教えるための入門版です。
<br><br>
迷蹤と拳路を省略し、下盤の軽功鍛練のみを抽出した浅い入門書です。

## 習得方法

-   2年目崆峒派留学－飛天門入門で習得
-   2年目崆峒派留学－玄功門入門時に飛天門秘笈を選択
-   2年目崆峒派留学－飛天門季試一勝（[虞小梅](/ja/people/characters/girl3)の好感度 < 25）
-   2年目10月中旬－四師兄の物品販売 ($1200)
