# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8107" />

全名を『黄帝八十一難経』という、著名な医学著作であり、『黄帝内経』の難問と要旨を解明することを目的としています。
<br><br>
著名な医者扁鵲の著作とされ、「五邪相干」「十変」理論を強調し、五臓生剋関係を解説します。
<br><br>
脈学、経絡、臓腑、さらには鍼法に関する論述は、後世の医学に深遠な影響を与えています。
