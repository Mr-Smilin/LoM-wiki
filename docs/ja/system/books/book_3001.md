# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="3001"></BookItemIcon>

宋太祖趙匡胤に因みて名付けられ、実戦の攻防を重視し、型は厳密で、足捌きが柔軟です。

身を縮めて猫のごとく、身体を揺らして虎のごとく、行き来は龍のごとく、動くことは稲妻のごとし。

太祖の名を冠しますが、実は嵩山拳法です。
