# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2004"></BookItemIcon>

崆峒・唐門の侠客が閉関して創出した合撃の術。
<br><br>
一人が拳を使い、一人が劍を使い、互いに補い合い、琴瑟和諧（がっせいわわ）のように、心が通じるほど威力は強くなる。
<br><br>
ただし、秘笈は持っているが、相手がいない。

## 入手方法

- 2年12月下旬－下山時に門人を召集する中で、別派の想い人である唐門女弟子を倒す（天命サイコロ名声補正、≦19が必要）
