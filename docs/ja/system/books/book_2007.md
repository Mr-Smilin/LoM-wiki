# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2007"></BookItemIcon>

峨嵋が劍派だった時代から伝えられた古い鍛身法。複雑さを簡潔にしながらも大らかで広々としており、その剛猛さながら峰のように折れ曲がるしなやかさがあり、余地を残す。
<br><br>
八路の斬法は一見平凡だが、実は高い武功を修得できるものではなく、ただ体を強健にし、筋骨を引き伸ばすのに大いに役立つ。
