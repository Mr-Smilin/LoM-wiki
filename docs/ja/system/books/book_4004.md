# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4004"></BookItemIcon>

武林盟主大俠龍淵七種の成名絶技の一つです。

玄玄如意、妙手空空で、素手で人の兵器を接することができ、人の衣囊に探ることができ、無往不利です。

龍淵少年時代にはこの招式を用いて人の銭財を扒くことが少なくありませんでしたが、彼が覚悟して脱皮し、一肩に武林の命運を担うまでになると、秘笈の中の盗物の手法を削除してしまいました。
