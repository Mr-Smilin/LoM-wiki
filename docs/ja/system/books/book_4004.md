# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="4004"></BookItemIcon>

武林盟主にして大俠龍淵の成名絶技七種の一つです。
<br><br>
玄玄如意、妙手空空にして、素手で敵の武器を受け止め、他人の衣嚢から物を盗むことができ、無敵の技です。
<br><br>
龍淵は少年時代、この招を使って他人の銭財を盗むことが少なくありませんでしたが、やがて目覚めて成長し、一肩に武林の命運を担うようになると、秘笈から盗術の手法を削除してしまいました。
