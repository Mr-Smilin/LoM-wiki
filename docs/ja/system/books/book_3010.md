# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="3010"></BookItemIcon>

一つの神奇な武功で、著者の氏名はすでに失われています。

掌法でもあり、剣法でもあり、手に剣がなければ、心に剣があります。

招数は精妙にして、一分の殺気もなく、自娯自楽のためのようで、舞わせれば清輝明月のようで、酣暢淋漓です。

## 習得方法

- 留学していない状態で、千面人魔事件を経て[龍淵](/ja/people/characters/special2)を認識し、[龍湘](/ja/people/characters/girl8)と縁を結ぶ時に信義について雑談して贈与されます。

## 備考

- [龍淵](/ja/people/characters/special2)と[王二壮](/ja/people/characters/special401)が酒に酔って互いに殴ることで創出された武功です。
- 習得時期はほぼ結局に近く、また大量の黄酒が必要であり、提前準備してこそ習成できるかもしれません。
