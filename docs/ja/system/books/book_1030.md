# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="1030"></BookItemIcon>

嵩山派の古典的な硬気功で、嵩山寺十八銅人の得意とする絶技。
<br><br>
金鐘罩は内外を兼修し、外は筋骨を壮健にし、内は臓腑を固める。功が成れば身は銅鉄のようで、拳脚がどれだけ加わっても痛みも痒みもない。

- 鉄布衫：体力に応じて防御力が上昇
- 金鐘罩：気≧20で使用可、気を10消費。防御コマンドが【金鐘罩】に変わり、実行後【護体気罩】を獲得。さらに内功の50%を防御に転換<br>
【護体気罩】効果－受けた刺突・絶招ダメージを75%～100%の比率で相手に反射、発動後に解除される
- 不壊金身：【戦役】で金剛罩を発動可能

## 入手方法

- 四書イベント
- 2年10月中旬－唐惟元から購入($3500)

## 備考

- 反射流には必須の招式で、医術の回復と組み合わせれば、敵を反射で倒すことができる
