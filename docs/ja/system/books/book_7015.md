# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7015" />

青城派の内功心法で、無為持修を主張し、守無致虚する。
<br><br>
心が清浄無為であれば、かえって有為となる。無心無念こそを大成とし、一切のイメージや観想を忌む。それでも道はおのずから生まれてくる。
<br><br>
習得も精通も難しいが、成った暁には計り知れない力となる。
