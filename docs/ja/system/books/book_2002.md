# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2002"></BookItemIcon>

福・建地域で活動する滄幫弟子が修練する武学。
<br><br>
滄幫の背後には上官世家が控えており、これは誰もが心に留める秘密だ。翻浪刀法さえ、上官家の伝承刀法から発展させたもので、船舵・船櫓で施展することで、却って隠しきれない。
<br><br>
この刀法を施展すれば、江上の波浪が猛烈に打ち寄せるようで、絶え間ない。弱者を相手なら陣陣と迫り、強者に対しては後ろを守る意志で臨む。敵が必死に抗すれば、空を切るしかなく、一たび隙が出れば、後の波が容赦なく迫ってくる。剛中に柔あり、柔中に剛あり、攻中に守あり、守中に攻あり、これは最上の刀法である。
<br><br>
ただし、船舵・船櫓で修練すれば、本道から外れることになる。

- 魚の卸売イベント－許大鯨が福韞に勝った後、金を払って購入($500)、または「買えない」を選んで強奪して獲得
