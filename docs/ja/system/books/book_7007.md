# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7007"></BookItemIcon>

巫山と長江流域の交差点にある鍾乳洞に隠されていた、壁に描かれた神秘的な武功です。その由来は不明で、どの朝代のものかも定かではありません。9枚の連環図で修練法を示しており、正邪両属性を兼ね備えた異常な修練法門で、通常の功法とは一線を画していますが、簡潔かつ明瞭で、目を開かせるような効果があります。
<br><br>
誤って水に落ちた漁夫がこれを発見した後、拓本にして朝廷に上納し、褒賞を得ました。しかし朝廷はこれを重視せず、この武功は官僚の書棚に埋もれたままになってしまいました。
<br><br>
その鍾乳洞も水位上昇により、完全に水没してしまいました。

## 入手方法

- 囲城イベント－外出して拳法を練る時の南宮深からの贈呈 (処世≧中庸)
- 3年目4月上旬－[唐惟元](/ja/people/characters/brother4)の買物($1500)

## 備考

- 南宮深からの贈呈時のテキスト：南宮深は君が師長の指導を受けていないことを知り、君が無理に内功を修練することで陰陽が乱れ、身体を傷つけてしまうことを心配しています。そこで君に一冊の内功図録『巫山洞府九寶圖』を贈りました。<br>南宮深は謙虚に言うには、この図を学んでも絶世の武功を身につけることはできないが、二儀を調整し、五行を温養することができるだろう、きっと君の役に立つはずだと。実は彼は君がこの図の貴重さを知ったら、簡単に受け取らないだろうと懸念していただけです。
