# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="3000"></BookItemIcon>

嵩山寺の僧侶が禅定と苦行に勤めるには、三つの不善根から遠ざかる必要があります。貪、嗔、癡は世間のすべての煩悩の根本であり、煩悩は人に種々の悪業を造らしめます。
<br><br>
解脱を得るためには、五蘊を知り、顛倒した夢想を遠離し、ようやく究竟涅槃に到達することができます。
<br><br>
しかし大道がここにあっても、世の中で果たして誰がこれを成し遂げることができるのでしょうか？
<br><br>
このとき、苦惱拳はまさに症状を治す良い方法です。拳を修練して汗を流し、疲れ果てれば、自然と雑念を抱く気力もなくなります。
