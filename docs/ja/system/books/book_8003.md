# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8003" />

著者は葉雲裳。字跡は非常に美しく、筆の動きには幼さが感じられ、彼女の人柄そのもののようです。<br><br>この書は雪合戦の技巧を記したもので、見事に《正攻》《奇襲》の2編に分けられています。この書に出てくる技のほとんどは彼女の想像から生まれたものです。<br><br>雪玉の中に石を隠すこと、毛虫を入れること、雪玉をズボンに押し込むなど、手段を選ばない残忍な伎倆を記載しています。
