# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2003"></BookItemIcon>

一代の儒俠・曾文先生の著した書で、四書五経と十三経をまとめ、君子は独善心を慎むべきことを論述しており、心から体に表れ、知識から礼儀に形づくられ、不遜でもなく卑下でもなく、言行一致している。月が替わり星が移ろうとも彼の所にしたがい、江山がどう易主しようとも頼んでくれ、自分は本心を守ったまま、自分を修養するのだ。
<br><br>
人間界がどれほど善くても、私はただ私であるだけ。世道がどれほど悪くても、私はただ私であるだけ。この身を捨てても、また然りだ。
<br><br>
凡そ劍を練り武を習う者は、本心を守るべきで、劍の両刃を鏡として、恥じることなく問い、謙虚にして自強止まず、困難に遭っても憂わず、宴遊の時にも心を引き締め、権豪に遭っても恐れず、これが君子の劍である。
