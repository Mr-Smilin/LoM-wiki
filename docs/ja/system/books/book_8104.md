# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8104" />

唐門の祖伝医書。当代唯一の毒を薬に用いる医書です。<br><br>苗疆と仏教学説を融合させ、一椀の水には8万4千の蟲がいると観て、蟲は毒であり、また五行六合に分けられ、五臓六腑に対応しています。多すぎると害をなし、少なすぎると病となり、尽きると虚となり、誤ると狂気となります。<br><br>人は必ずしも五毒に害されるわけではなく、実は毒性がまだ発動していないだけです。ただ薬で養うことにより、その心意に従い逆らわず、体の毒を自身の用に供すれば、百病寄せ付けません。
