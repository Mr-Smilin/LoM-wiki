# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8108" />

古今流行の折り紙の方法を集めたもので、紙箱、紙船、紙蛙、紙鶴、紙魚、紙花、紙剣から紙の登雲梯まで掲載されています。
<br><br>
図解が精密で、解説が明確で、この本があれば、誰もが折り紙の達人になることができます。

## 入手方法

- 留学をせず、荒れた寺院ルートで唐門に通報し、講経堂のルーレットで【摺紙集】を入手する

## 備考

- [掌門](/ja/people/characters/master)こと唐中翎が[四師兄](/ja/people/characters/brother4)こと唐惟元に預けて、[小師妹](/ja/people/characters/girl0)こと唐默鈴に贈った秘笈です。
- 唐默鈴は大変喜び、本の中のすべてのものを折りました。色紙がなくなると、悪知恵を働かせて秘笈を折ってしまい、本の皮は床の下に隠しました。自分の秘笈を折り終わると、他人の秘笈まで折ってしまいました。最終的に事件が発覚し、唐默鈴は掌門の家法で懲罰され、本も封印されてしまいました。
- 『摺紙戦士』は台湾の漫画家周顯宗の漫画作品です。
