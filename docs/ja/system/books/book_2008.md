# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="2008" />

青城の秘笈で、『奇門遁甲』の八門方位・星象・地形に基づいて作られた個人の陣法。
<br><br>
この陣法を解きほぐして簡略化し、刀技に溶け込ませたものが八門鎖金刀であり、その精妙さは並ぶものがない。
