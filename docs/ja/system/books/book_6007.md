# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="6007"></BookItemIcon>

江湖一の美女奪魄幽蘭の成名絶技です。
<br><br>
奪魄門の暗襲の術と雪山派の内功を揉合わせ、身法は鬼魅の如く、人の魂魄を勾引することができます。
<br><br>
摧心、蝕骨、銷魂、奪魄の四式は、一招が一招より激しく、極めて毒辣であり、これを習うと恐らく天和を傷つけてしまうでしょう。
