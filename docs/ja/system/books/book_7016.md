# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="7016" />

道門の虎嘯功。猛虎の咆哮に譬えて、修道の人が勇猛精進し、猛虎のように威猛で、群邪を辟易させることを形容する。
<br><br>
この功は丹田の内気を外に発することで、人と対戦する際に、発功して呼嘯を放つ。声は雷霆のごとく千里に響き渡り、人の心魄を奪い、甚だしくは内傷さえ与える。

- 虎嘯功：境界二以上。口功が「虎嘯吼」に変わり、追加の内功を消費して、口功に実質的なダメージを持たせる。

## 入手方法

青城留学で、以下の条件を同時に満たした後、9月上～11月中に申屠龍から贈られる：
1. 9月下旬までに、3回修練した後に排擠イベントが発動。三五斬妖劍を獲得した後、これを練満する。
2. 申屠龍≧15。修練時に一定確率で彼と切磋できる。最低1回は必要。
