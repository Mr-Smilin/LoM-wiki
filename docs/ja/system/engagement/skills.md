### スキル
<div style="margin-top: 5px;"></div>
ゲームをプレイしている間に、異なるスキルや特殊スキルを学びます。戦闘で効果を発揮するスキルはすべてこのエリアに表示されます<br>
マウスを上にかざすと説明を確認できます。ぜひ活用してください<br>
スキルの下にダイヤモンド形のボックスがある場合、そのスキルの使用に制限があることを意味します<br>
使用可能回数、または残りターン数を表す場合があります<div style="margin-top: 5px;"></div>
<img class='guide-img' src='/images/engagement/user-left.webp' alt='残り使用回数'>


### サポート
<div style="margin-top: 5px;"></div>
時々、戦闘にサポートキャラが表示されることがあります<br>
その戦闘限定の特殊スキルと見なせます<br>
マウスをキャラクターの画像に移動すると、同様に説明が表示されます<br>
