# 用語対訳表 (繁體中文 / 日本語 / English)

本ページは活俠傳 (カッキョウデン / Legend of Mortal) 日本語版 wiki の訳語基準表です。
※「カッキョウデン」は開発元 Obb Studio 公認の日本語読みです。

- **訳語の優先順位**: ① 日本語化 MOD ([LOM-JPMOD](https://dlaqe2334.github.io/LOM-JPMOD/) / [GitHub](https://github.com/Dlaqe2334/LOM-JPMOD)) の既存訳 → ② 日本の武侠翻訳慣行に沿った訳語
- 表中の **★MOD** は MOD の翻訳データ (Stringtable.csv) で確認済みの訳語です
- English 列は本 wiki [英語版](/en/) での表記です (英語版に対応が無い語は空欄)
- **表記ルール**: 記事本文では固有名詞の初出時に「日本語訳 (原文中国語)」を併記します。原文と日本語訳が同一表記の場合は併記しません
- 活俠傳に公式日本語版はありません。中国語版・日本語化 MOD 版のどちらでプレイしていても照合できるよう、原文表記を必ず残しています
