## {{ $frontmatter.title }}

裏山、正心堂、講経堂、大門を除くすべての建築物に初出時のキャラクターイベントがあります。カーソルを建築物に移動すると操作できるようになります。

<br>

江湖は危険で、唐門の中さえも江湖の縮図です。初期段階では迅速に実力を蓄積する必要があるため、修練場で武功を練習しましょう。秘笈を手に入れたら裏山に行って鍛練し、門派レベルが2になった後、鍛冶場で護心鏡（2級護甲）を作成することができます。

<br>

それ以外にも、各建築物にはそれぞれ独自のイベントがあります。あちこち探索して、こまめにセーブしましょう！
