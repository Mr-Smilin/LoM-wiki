## 条件とフロー

<strong>条件：</strong><Girl8Icon>龍湘</Girl8Icon>以外の者と婚約、<Girl8Icon>龍湘</Girl8Icon>が唐門に加入、想い人が<Girl8Icon>龍湘</Girl8Icon>、門人解散<br>

1. 第三年十月中旬の武林大会で、錦香宮が暴露された後、<Girl8Icon>龍湘</Girl8Icon>が唐門に身を寄せる時、**📖「世間の冷たい視線を無視して龍湘を迎える」**を選択、<Girl8Icon>龍湘</Girl8Icon>が唐門に加入。

3. 敵と隠居について投票を実施：[唐惟元](/ja/people/characters/brother4)が帰還した場合、[唐惟元](/ja/people/characters/brother4)の投票が追加される。唐默鈴と婚約した場合、<Girl0Icon>唐默鈴</Girl0Icon>の投票が追加される。
   + 主人公の決策：
     + **📜議題「決策：敵と戦う」**：敵と戦う＋１票。
     + **📜議題「決策：江湖から隠居」**：江湖から隠居＋１票。
   + **🎲運命：三師兄の決策......**(上限70、➕加値：名声)：
     + **🧾【＜60】門弟を解散**：江湖から隠居＋１票。
     + **🧾【≧60】敵と戦う**：江湖から隠居＋１票。
   + **🎲運命：四師兄の決策......**(上限99、➕加値：名声)：
     + **🧾【＜80】門弟を解散**：江湖から隠居＋１票。
     + **🧾【≧80】敵と戦う**：江湖から隠居＋１票。
   + **🎲運命：小師妹の決策......**(上限99、➕加値：<Girl0Icon>唐默鈴</Girl0Icon>好感度*0.1)：
     + **🧾【＜30】紙が鶴に折られた**：無効票。
     + **🧾【＜70】江湖から隠居**：江湖から隠居＋１票。
     + **🧾【≧70】師兄と同じ**：主人公の票数＋１。
4. 以下の条件を満たして、門弟を解散することを決定：
   + 江湖から隠居の票が多い。
   + 票が同数で主人公が江湖から隠居を選択し、🗡️[唐陞](/ja/people/characters/brother3)との決闘に勝利。
   + 票が同数で[唐陞](/ja/people/characters/brother3)が江湖から隠居を選択し、🗡️[唐陞](/ja/people/characters/brother3)との決闘に敗戦。
5. 後山へ向かい、**📖選択肢「湘姉、一緒に来ませんか？（心を明かす）」**(選択肢を開くには想い人が<Girl8Icon>龍湘</Girl8Icon>である必要があります)。
6. 既に他のキャラと婚約している場合、🗡️<Girl8Icon>龍湘</Girl8Icon>が決闘を仕掛け、勝利後に**📖選択肢「君を手放すことはない」**を選択、エンディングに進む。
