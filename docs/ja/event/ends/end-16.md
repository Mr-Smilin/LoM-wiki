> 書き下し文：周の穆王、南に征きて、一軍盡く化け、君子は猿と為り鶴と為り、小人は蟲と為り沙と為る。
>
> 大意：西周の穆王が南方へ遠征した際、軍勢全体が異形へと化した。身分の高い者（君子）は猿や鶴に、身分の低い者（小人）は虫や砂になったという。出典は『抱朴子』（東晋・葛洪）を引く『太平御覽』巻九一六・羽族部・鶴。
