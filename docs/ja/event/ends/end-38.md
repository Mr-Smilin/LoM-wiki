# {{ $frontmatter.title }}

<EndBackground no=38 title="神州の旅人">
お前たちの旅は、出発したその日から止まることなく、<br>
行く先々で遊び尽くした。彼女が金を出し、お前が命を<br>
張り、旧友を訪ね、去るも留まるも惜しまず、神州大地<br>
を巡り、その足跡は崑崙山で途絶えた。<br>
