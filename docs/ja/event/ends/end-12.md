## 条件と進め方

<strong>条件：なし。</strong><br>
**進め方：**<br>
1. 第3年3月期間、練功場または後山に5回訪れ、[唐惟元](/ja/people/characters/brother4)が商隊参加を招待するイベントが発生し、2番目の選択肢で**📖抉擇：【よろしい。】**を選ぶ
2. 想い人が小師妹でなく、あるいは小師妹好感度＜30の場合、エンディングに進む。

## 分岐と変化

### キャラクターの属性
以下の条件を満たすかどうかが招待イベント部分の対話に影響し、伝説文本に影響しない
+ 処世≦40【知礼、気難しい】
+ 道徳≦40【悪人、悪党】
+ 処世≧60【豪爽、粗野】
+ 品性≦40【短気、狂気】
+ 学問≧45
+ 道徳≦40【悪人、悪党】かつ品性＞40【中庸、沈着、君子】：招待イベント中に1000銀両を獲得。


### 保有銀両
銀両≧10000を満たすかどうかが招待イベント部分の対話に影響し、伝説文本に影響しない。

### 四師兄の好感度
[唐惟元](/ja/people/characters/brother4)好感度≧45の場合、[唐惟元](/ja/people/characters/brother4)が自ら商隊への参加を招待。好感度＜45の場合は三師兄の託しで招待され、伝説文本に影響しない。

### 第一の選択肢
選択肢後の対話に影響し、伝説文本に影響しない。

### 大師兄と二師兄の好感度
伝説文本に影響しない。
+ [唐布衣](/ja/people/characters/brother1)好感度≧45：[唐布衣](/ja/people/characters/brother1)が生前相声役者になりたかったことを知る。
+ [唐布衣](/ja/people/characters/brother1)好感度≧40かつ[唐布衣](/ja/people/characters/brother1)好感度≧[唐錚](/ja/people/characters/brother2)好感度：対話で[唐布衣](/ja/people/characters/brother1)に憧れを抱く。
+ [唐錚](/ja/people/characters/brother2)好感度≧40かつ[唐錚](/ja/people/characters/brother2)好感度＞[唐布衣](/ja/people/characters/brother1)：対話で[唐錚](/ja/people/characters/brother2)に憧れを抱く。
+ [唐布衣](/ja/people/characters/brother1)好感度＜40かつ[唐錚](/ja/people/characters/brother2)好感度＜40：対話で[唐中翎](/ja/people/characters/master)掌門に憧れを抱く。

### 想い人
伝説文本に影響しない。
+ 小師妹：招待イベント部分の対話に影響し、エンディング前に小師妹と別れの場面が追加される。第2年2月上旬に小師妹と「離開するときに先に知らせる」と約定したイベントがあった場合（小師妹好感度≧40が必要）、別れの場面で追加セリフが出現する。
+ [上官螢](/ja/people/characters/girl4)：招待イベント部分の対話に影響する。
