## フロー

青城留学期間中、活動を実施して功績ポイントを累積します。計点は以下の通り：
+ 上清宮修真 +1
+ 上清宮道士 +1<br>
(地獄道と誤認時+0、鬼卒助陣で五ターン対戦+1)
+ 青城山鎮医館－青城弟子求診 +2
+ 常風観散歩－吟唱、悟ったような、路人侠に負ける -1

二年目九月下旬に鄒博が帰来した後、上清宮を訪問。功績ポイント≥6の場合、褒賞イベントが発動し、【青城派に入門】を選択。
