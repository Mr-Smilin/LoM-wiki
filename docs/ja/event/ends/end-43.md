# {{ $frontmatter.title }}

<EndBackground no=43 title="寵愛を失う">
子どもが生まれると、夏侯蘭の冷たい心はついに溶けた。しかし<br>
あなたの寵愛は失われた。<br>
彼女は家庭に専念し、しばしば子どもが自分に似て<br>
あなたに似ていないことを幸いに思い、あなたが子どもを打つことを許さず、極度に甘やかす。<br>
子どもに良い教育を受けさせるため、ある日外出して活捉して<br>
魏菊を家に連れ帰り、家庭教師とした。
</EndBackground>

1. 二年二月上旬唐門会議、**📜議題「崆峒派へ留学したい」**。
2. 二年三月上旬、門派抽籤に参加し奪魄門に加入してから、<Girl5Icon>夏侯蘭</Girl5Icon>を認識する
   + 二番目の選択**📖選項「門派の名誉のために同意」**。
   + 二番目の選択**📖選項「他派に敵視される覚悟で反対する。」**、三番目の選択**📖選項「構わない、自分も抽籤に参加すればいい、魏掌門の手数を減らすこともできる」**。
3. **🎲運命：門派抽籤の結果......**(上限99)：**🧾【＜75】奪魄門**を出す、奪魄門に加入。
4. 二年四月上旬、襲撃される、**🎲運命：結果......**(上限90、➕加値：拳掌)：**🧾【＞50】防禦成功**を出す、🗡️峨嵋弟子との決闘勝利、🗡️再び峨嵋弟子と決闘敗北または四回合終了で戦闘終了。

6. 三年六月上旬唐門会議、**📜議題「雪山行」**。
7. 三年七月中旬<Girl5Icon>夏侯蘭</Girl5Icon>に穴を塞がれた後、連続三回**📖「抵抗」**を選択（二番目の選択肢開放には穴を塞がれる前に身に着けている道具に酒があって<Girl5Icon>夏侯蘭</Girl5Icon>を酔わせることが必要、三番目の選択肢開放には想い人が<Girl5Icon>夏侯蘭</Girl5Icon>であることが必要）、最初の三回の運命転盤の結果は関係ない。
