## 条件と流れ
<strong>条件：</strong><Girl5Icon>夏侯蘭</Girl5Icon>と結緣<br>
**流れ：**<br>

### １、夏侯蘭との結緣
1. 2年目2月上旬の唐門会議で、**📜議題：崆峒派に留学したい**を選ぶ。
2. 2年目3月上旬、門派くじを行う時：
   + 2番目の選択肢で**📖選択：門派の名声のため、同意する**。
   + 2番目の選択肢で**📖選択：他派に敵視される覚悟で反対する**、3番目の選択肢で**📖選択：構わない、私がくじに加わるから、魏掌門に迷惑をかけずに済む**。
3. **🎲運命：門派くじ結果……**(上限99)：**🧾【＜75】奪魂門**が出て、奪魂門に入門。
4. 2年目4月上旬、不意打ちを受ける。**🎲運命：結果……**(上限90、➕加値：拳掌)：**🧾【＞50】防御成功**、🗡️峨嵋派弟子との決闘で勝利、🗡️再度峨嵋派弟子と決闘で敗戦、または4回目ラウンド終了で戦闘終了。
5. 森に逃げ込んだ後、最初の選択肢の後、森から出て🗡️峨嵋派弟子と再度決闘、勝敗は問わない。2番目の選択肢の後、3番目の選択肢で**📖選択：師礼で感謝をささげる**を選択、<Girl5Icon>夏侯蘭</Girl5Icon>に師事。
6. 3年目6月上旬の唐門会議で、**📜議題：雪山行**。
7. 3年目7月中旬、<Girl5Icon>夏侯蘭</Girl5Icon>に経穴を塞がれた後、連続3回**📖選択：抵抗する**を選ぶ(2番目の選択肢開放には経穴を塞がれる前に酒を持っていて<Girl5Icon>夏侯蘭</Girl5Icon>に酔わせることが必要、3番目の選択肢開放には想い人が<Girl5Icon>夏侯蘭</Girl5Icon>であることが必要)。最初の3回の運命ルーレット結果は重要ではない。
8. 4回目の運命ルーレット**🎲運命：お前の抵抗……**(上限50、➕加値：<Girl5Icon>夏侯蘭</Girl5Icon>好感度)：**🧾【＝99】抵抗**が出る。
9. 🗡️<Girl5Icon>夏侯蘭</Girl5Icon>との決闘で勝利、<span style='color: #FF1493;'><Girl5Icon>夏侯蘭</Girl5Icon>と結緣</span>、3年目8月上旬に【夏侯蘭の髪帯】を獲得。

### ２－１、抵抗を続けるルート
1. 3年目12月上旬の最後の行動前に、[衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)で抵抗を続けるシナリオに進む。
2. 唐門包囲戦：[衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)の結果により開始時期が異なる
   + [衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)が1️⃣西武林盟成立を選択：4年目2月中旬に行動が2回残っている時に開始。
   + [衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)が2️⃣西武林盟成立が叶わなかった場合：4年目1月中旬に行動が2回残っている時に開始。
3. **🎲運命：弟子離散の結果……**(上限50、➕加値：向心/2)：結果は影響しない、**📖選択：私は……お前の言葉に従う**を選んで、エンディング進入。

### ２－２、西武林盟ルート
1. 3年目12月上旬に最後の行動が残っている時、[衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)で**西武林盟**シナリオに進む。
2. 4年目2月中旬に行動が2回残っている時、西武林大会後に[東西武林盟會戰](/ja/event/detailed_description/4-02-2-東西武林盟會戰)が始まり、**📜議題：戦役：唐門出撃**。
3. 途中のプロセスは複数の組み合わせがあり、ほとんど影響しないが、以下の条件を満たす場合は特別な要件あり：
   + 武林大會が錦香宮を支援していない、[丹霞子](/ja/people/characters/special404)が唐門に入門していない、**📖選択：門を堅く閉ざす**時、内功≧20かつ性情≧20が必要。
     + 上記に加えて、<Girl5Icon>夏侯蘭</Girl5Icon>と結緣している場合、🗡️[萬里鵬程](/ja/people/characters/special6)との決闘で勝利が必要。
4. 最初の⚔️戦役の勝敗は問わない、**📖選択：外堡を全力で守る**、⚔️戦役勝利、**📖選択：私は……お前の言葉に従う**を選んで、エンディング進入。
