## 条件と手順

<b>条件：</b>崆峒派に留学、虞小梅が唐門に加入、想い人：郁竹、郁竹好感度≧45<br>
<b>手順：</b><br>
1. 第2年2月上旬唐門会議で**📜議題：(事件)崆峒派に留学したい**を選ぶ．
2. 第2年10月下旬、虞小梅好感度≧20を満たす：<span style='color: Yellow;'>虞小梅が唐門に加入</span>．
4. 第3年1月中旬、[唐布衣](/ja/people/characters/brother1)と一緒に下山しない：
   + 🎲**天命：幾人の援手が必要か？....**(上限60、➕加値：[唐布衣](/ja/people/characters/brother1)好感度）：
     + **🧾【＜80】必要ない**．
     + **🧾【≧80】必要ない、俺と趙活で十分だ。**→**📖議題：嫌だよ(猛烈に抵抗)**．
5. 第3年、[唐布衣](/ja/people/characters/brother1)が重傷で帰ってくる。医術スキル≧5または総毒薬ポイント≧、以下の選択肢の組み合わせで[唐布衣](/ja/people/characters/brother1)を治療する：
   + **📖抉擇：決意**．
   + **📖抉擇：躊躇**→**📖抉擇：大師兄は死なせられない**→**📖抉擇：決意**．
   + **📖抉擇：躊躇**→**📖抉擇：大師兄は死なせられない**→**📖抉擇：躊躇**→**📖抉擇：大師兄は死なせられない**→**📖抉擇：決意**．
   + **📖抉擇：躊躇**→**📖抉擇：大師兄は死なせられない**→**📖抉擇：躊躇**→**📖抉擇：大師兄は死なせられない**→**📖抉擇：躊躇**．
6. 虞小梅が登場した後の選択肢で**📖抉擇：心を鎮める**を選ぶ．
7. **想い人：郁竹**、郁竹好感度≧45、**📖抉擇：俺の家を見に来ないか？**を選んでエンディングに進む．
