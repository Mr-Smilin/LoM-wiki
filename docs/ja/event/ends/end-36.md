## 条件と流れ
<strong>条件：</strong>抵抗を続けるシナリオに進む。<br>
**流れ：**<br>
1. 3年目12月上旬の最後の行動前に、[衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)で抵抗を続けるシナリオに進む。
2. 唐門包囲戦：[衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)の結果により開始時期が異なる。
   + [衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)が1️⃣西武林盟成立の決策：4年目2月中旬に行動が2回残っている時に開始。
   + [衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)が2️⃣西武林盟不成立の決策：4年目1月中旬に行動が2回残っている時に開始。
3. **🎲運命：弟子離散の結果……**(上限50、➕加値：団結/2)：
   + **🧾【＝99】誰も去らなかった**：門人-0。
   + **🧾【＞60】少数が去った**：門人-5。
   + **🧾【＞30】大多数が去った**：門人-20。
   + **🧾【≦30】ほぼ全員去った**：門人ゼロ、以降の2つの⚔️乱戦が🗡️決闘に変化。
4. **🎲運命：最初の戦……**(上限99)：
   + **🧾【＞60】100人いた**：
     + 前の結果が**🧾【≦30】ほぼ全員去った**：🗡️武林盟の豪傑たちと決闘、勝利が必要。
     + その他：⚔️乱戦、勝利が必要。
   + **🧾【＞30】3人の高手**：相手が誰であれ、🗡️決闘で勝利が必要。
   + **🧾【≦30】絶頂の高手1人**：🗡️[萬里鵬程](/ja/people/characters/special6)と決闘、勝利が必要。
5. 山を駆け下りて決戦を挑む：
   + 前の結果が**🧾【≦30】ほぼ全員去った**：🗡️武林盟の豪傑たちと決闘、勝利が必要
   + その他の結果：
     + **📖選択「全力突破」**：⚔️乱戦、勝敗は問わない。
     + **📖選択「一人も生き残すな」**：⚔️乱戦、勝利が必要→⚔️乱戦、勝敗は問わない。
6. 🗡️武林盟主-[瑞笙](/ja/people/characters/special999)と決闘、敗戦後、エンディング進入。
