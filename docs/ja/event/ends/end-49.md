## 条件とフロー
<b>条件：</b>西武林大会を開催、[南宮深](/ja/people/characters/special102)好感度≧20。<br>
<b>フロー：</b><br>
1. 第三年十二月上旬に1つの行動が残った時、[衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)で西武林盟シナリオラインへ進入。
2. 第四年二月中旬に2つの行動が残った時、西武林大会を開催。
3. 東西武林盟会戦：
   + **📜議題：決鬥：三師兄を護衛**：
     1. 2つの🗡️決鬥に勝利。
     2. **🎲天命：武林盟の消耗......**(上限60、➕➖加減値：双方の団結力と支援勢力)：**🧾【＜90】激しい抵抗に遭遇、損害甚大** または **🧾【＜45】損害軽微、西武林盟を徹底的に蹂躙**。
     3. ⚔️戦役で敗北時に[南宮深](/ja/people/characters/special102)好感度≧20なら、エンディングへ。
   + **📜議題：戦役：唐門出撃**、
     1. 外堡を防守、過程は多数の組み合わせがあり大部分に影響なし。ただし以下の条件を満たす場合は特別要件あり：
        + もし[武林大会](/ja/event/simple/1-12-1-南宮壽宴)で錦香宮を支持しておらず、[丹霞子](/ja/people/characters/special404)が唐門に参入していない場合、**📖選択：大門を固く閉ざす**時に、内功≧20かつ性情≧20が必須。
          + これに加えて、<Girl5Icon>夏侯蘭</Girl5Icon>と結緣していれば、🗡️[萬里鵬程](/ja/people/characters/special6)との決鬥で勝利必須。
     2. ⚔️戦役の勝負は関係なし、**📖選択：兵を二手に分けて、一方は外堡を固守しつつ周旋し、一方は大院へ上がり決戦準備：**。
     3. **🎲天命：大院の情勢......**(上限60、➕加算値：団結力と支援勢力、➖減値：敵方の団結力)：
        + **🧾【＜30】唐門の大勢は既に去った** または **🧾【＜66】戦況激化**：
          1. **🎲天命：お前の対手......**(上限99、➕加算値：[南宮深](/ja/people/characters/special102)好感度、➖減値：<Girl4Icon>上官螢</Girl4Icon>好感度)：**🧾【＜70】南宮深（鳳凰肩甲）**。
             * ※<Girl5Icon>夏侯蘭</Girl5Icon>または<Girl8Icon>龍湘</Girl8Icon>と結緣していれば、この段階をスキップ。
          2. 🗡️[南宮深](/ja/people/characters/special102)との決鬥で敗北時に[飛俠歸來](/ja/event/detailed_description/4-02-2-飛俠歸來)を満たさず、且つ[南宮深](/ja/people/characters/special102)好感度≧20なら、エンディングへ。
