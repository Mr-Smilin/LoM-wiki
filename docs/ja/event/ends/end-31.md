### 一、夏侯蘭と婚約
1. 第二年二月上旬唐門会議で、**📜議題「崆峒派留学に行きたい」**。
2. 第二年三月上旬、門派くじに参加が必要。奪魂門に加入することで<Girl5Icon>夏侯蘭</Girl5Icon>と知り合える。
   + 第二の選択肢で**📖選択肢「門派の名声のため同意」**を選択。
   + 第二の選択肢で**📖選択肢「他派に敵視される覚悟で反論する」**、第三の選択肢で**📖「構わない、俺も一緒にくじを引く。魏掌門を困らせる必要もない」**を選択。
3. **🎲運命：門派くじの結果......**(上限99)：**🧾【＜75】奪魂門**を獲得、奪魂門に加入。
4. 第二年四月上旬、偷襲を受ける。**🎲運命：結果......**(上限90、➕加値：拳掌)：**🧾【＞50】防御成功**、🗡️峨嵋弟子との決闘に勝利、🗡️再び峨嵋弟子との決闘に敗戦するか、第四ターンで終了する。
5. 森に逃げた後、最初の選択肢後、🗡️森を出て峨嵋弟子と再度決闘、勝負は無関係。第二の選択肢後、第三の選択肢で**📖「師の礼で拝謝」**を選択、<Girl5Icon>夏侯蘭</Girl5Icon>に師事。
6. 第三年六月上旬唐門会議で、**📜議題「雪山行」**。
7. 第三年七月中旬に<Girl5Icon>夏侯蘭</Girl5Icon>に穴を指された後、連続三回**📖選択肢「抵抗」**を選択(第二の選択肢を開くには、穴を指される前に身に付けている道具に酒があって、それで<Girl5Icon>夏侯蘭</Girl5Icon>を酔わせる必要があります。第三の選択肢を開くには想い人が<Girl5Icon>夏侯蘭</Girl5Icon>である必要があります)。前三回の運命転盤の結果は重要ではありません。
8. 四回目の運命転盤**🎲運命：君の抵抗......**(上限50、➕加値：<Girl5Icon>夏侯蘭</Girl5Icon>好感度)：**🧾【＝99】抵抗**を獲得。
9. 🗡️<Girl5Icon>夏侯蘭</Girl5Icon>との決闘に勝利、<span style='color: #FF1493;'><Girl5Icon>夏侯蘭</Girl5Icon>と婚約</span>。第三年八月上旬に【夏侯蘭の髪帯】を獲得。

2. 敵と隠居について投票を実施：[四師兄](/ja/people/characters/brother4)が帰還した場合、[四師兄](/ja/people/characters/brother4)の投票が追加される。
   + 主人公の決策：
     + **📜議題「決策：敵と戦う」**：敵と戦う＋１票。
     + **📜議題「決策：江湖から隠居」**：江湖から隠居＋１票。
   + **🎲運命：三師兄の決策......**(上限70、➕加値：名声)：
     + 獲得**🧾【＜60】門弟を解散**：江湖から隠居＋１票。

   + ※門人解散エンディング結婚優先順位(v1.0.3200.2)：<Girl0Icon>唐默鈴</Girl0Icon>＞<Girl2Icon>葉雲裳</Girl2Icon>＞<Girl4Icon>上官螢</Girl4Icon>＞<Girl3Icon>虞小梅</Girl3Icon>＞<Girl6Icon>郁竹</Girl6Icon>＞<Girl7Icon>魏菊</Girl7Icon>＞<Girl5Icon>夏侯蘭</Girl5Icon>

## 分岐と変化

### 前往地点
大門を出る前に前往地点があり、伝説テキストに加わる。一部の地点のコンテンツは想い人と唐門に加入した成員によって異なる。

### 四師兄が帰還
[四師兄](/ja/people/characters/brother4)がエンディング前のプロット内に登場した場合、エンディング前のシーンで登場するが、伝説テキストに影響しない。

### 唐門に加入した成員
以下の成員が唐門に加入した場合、エンディング前のプロット内に登場し、伝説テキストに影響しない。
+ [福韞](/ja/people/characters/special206)：武林大会に向かう途中に[覚醒](/ja/people/characters/monk2)に嵩山への訪問を約束したかどうかによって、異なるセリフがある。
+ [樊嘯天](/ja/people/characters/special4)：[樊嘯天](/ja/people/characters/special4)が<Girl5Icon>夏侯蘭</Girl5Icon>に師事しているかどうかによって、異なるセリフがある。
