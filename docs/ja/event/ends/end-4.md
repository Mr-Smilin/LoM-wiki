# {{ $frontmatter.title }}

<EndBackground no=4 title="唐門の裏切り者">
あなたは唐門を裏切った。生存のため、栄華のためと言うが、<br>
心の奥底では、血よりも濃い<br>
師弟の絆を裏切ったことを知っている。<br>
<br>
<br>
</EndBackground>

## 条件
1. 一年四月中～二年一月下、貢献度<-150、怠けすぎて唐門から追放される。
2. 二年間は住む場所がなく廃寺に住んでいる、二年8月中旬に道徳が悪人以下に、【俺たちが唐門を攻撃しろ！】を選択。二連続の戦闘に全て勝利して、掌門に追放される。
