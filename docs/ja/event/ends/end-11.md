## 条件と流程

<b>条件：</b>崆峒派に留学、虞小梅が唐門に加入、心上人が郁竹でないか郁竹好感度＜45<br>
<b>流程：</b><br>
1. 第2年2月上旬唐門会議で**📜「崆峒派に留学したい」**を選ぶ．
2. 第2年10月下旬、虞小梅が唐門に加入(虞小梅好感度≧20が必要)．
4. 第3年1月中旬、[唐布衣](/ja/people/characters/brother1)と一緒に下山しない．
5. 第3年1月下旬、[唐布衣](/ja/people/characters/brother1)が重傷で帰ってくる。[唐布衣](/ja/people/characters/brother1)の治療を選ぶ(医術スキル≧5または総毒薬ポイント≧250が必要)．
6. 虞小梅が登場した後の選択肢で**📖「心を鎮める」**を選ぶ．
7. 心上人が郁竹でないか郁竹好感度＜45、エンディングに進む．

### 大師兄と一緒に下山しない条件説明：
+ **🎲天命：幾人の援手が必要か？....**(上限60、➕加値：[唐布衣](/ja/people/characters/brother1)好感度）：骰出**🧾【＜80】必要ない**．
+ **🎲天命：幾人の援手が必要か？....**(上限60、➕加値：[唐布衣](/ja/people/characters/brother1)好感度）：骰出**🧾【≧80】必要ない、俺と趙活で十分だ。**、後で**📖「嫌だよ(猛烈に抵抗)」**を選ぶ．

### 大師兄の治療選択肢説明：
毎回躊躇の対話が異なる．
+ **📖「決意」**を選ぶ．
+ **📖「躊躇」**→**📖「大師兄は死なせられない」**→**📖「決意」**．
+ **📖「躊躇」**→**📖「大師兄は死なせられない」**→**📖「躊躇」**→**📖「大師兄は死なせられない」**→**📖「決意」**．
+ **📖「躊躇」**→**📖「大師兄は死なせられない」**→**📖「躊躇」**→**📖「大師兄は死なせられない」**→**📖「躊躇」**．

## 分岐と変化

+ 性情≧60【勇敢、莽夫】：エンディング前部の対話に影響．
+ 道德＜40【悪人、悪党】：エンディング前部の対話に影響．
+ 修養＜60【中庸、短気、狂気】：エンディング前部の対話に影響．
