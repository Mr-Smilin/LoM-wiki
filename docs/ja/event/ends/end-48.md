# {{ $frontmatter.title }}

<EndBackground no=48 title="武林の伝説">
耕陽読書斎がお前の事跡を書物に編さんし、市場へ発行した。<br>
唐門の活俠の大名は一朝にして中原に広がり、知らぬものなし。<br>
お前の名は今この時より、伝説となり、万人に敬われる。<br>
<br>
</EndBackground>

## 条件とフロー

<b>条件：</b>西武林盟が成功裏に成立<br>
<b>フロー：</b><br>
1. 第三年十二月上旬の最終行動前。[衆人の決策](/ja/event/detailed_description/3-12-1-眾人的決策)で西武林盟シナリオラインへ進入。
2. 第四年三月上旬、眉山決戦で、🗡️武林盟主-[瑞笙](/ja/people/characters/special999)との決闘で勝利後、エンディングへ。

## 分岐と変化

### 戦闘後の選択肢

伝奇テキストは戦闘後の選択肢で変わる。

-   **📖「これほど悪い事をしといて、体裁など？死ね！」**を選択：[瑞笙](/ja/people/characters/special999)死亡、黒幕化なし。
-   **📖「誰かが責任を取らねばならん。」**、または**📖「遺言があるか？」**、さらに**📖「お前は選択をした、その責任を負うべきだ。」**を選択：[瑞笙](/ja/people/characters/special999)死亡、黒幕化あり。
-   **📖「遺言があるか？」**を選択してから、**📖「罪を功で贖うべきだ。」**を選択：[瑞笙](/ja/people/characters/special999)生存。

### 主人公のエンディング

-   **📖「これほど悪い事をしといて、体裁など？死ね！」**を選択、道德＜40【悪人、悪党】：主人公は武林盟主にならず、エンディング前のシナリオは盟主になった場合と大きく異なる。

### 大師兄の生存

[唐布衣大師兄](/ja/people/characters/brother1)が生存していれば、エンディング前のシナリオに登場する。

### 四師兄の帰還

[四師兄](/ja/people/characters/brother4)が以前に帰還していれば、エンディング前のシナリオに登場する。
