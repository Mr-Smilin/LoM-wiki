## 条件と流程

<b>条件：</b>崆峒派に留学<br>
<b>流程：</b><br>

1. 2年目2月初旬の行動前－唐門会議で、**📜 議題：(イベント)崆峒派への留学に行きたい**を選択。
2. 2年目5月下旬の行動前－崆峒季試：
    1. **📖 抉擇：挑戦する**
    2. 第1試合。**🎲 天命：勝敗……**(上限99)：
        - **🧾【＜50】南宮淺（ナンキュウセン）の勝利**：**🆚 決闘：南宮淺**で勝利が必要。
        - **🧾【≧50】南宮淺の敗北**：**🆚 決闘：青城弟子**で勝利が必要。
    3. 第2試合。**🎲 天命：次の対手は……**(上限99)：
        - **🧾【＜25】火法を使う剣客**：**🆚 決闘：点蒼弟子**で勝利が必要。
        - **🧾【＜50】暗算の技を習った僧**：**🆚 決闘：嵩山弟子**で勝利が必要。
        - **🧾【＜75】精良な兵器を装備した女俠**：**🆚 決闘：錦香弟子**で勝利が必要。
        - **🧾【≧75】心修養性を養った戦士**：**🆚 決闘：峨嵋弟子**で勝利が必要。
    4. 第3試合。**🎲 天命：次の対手は……**(上限99)：**🧾【＜33】丐幫の狂犬**：**🆚 決闘：樊嘯天（ハンショウテン）**で敗北が必要。
    5. **🎲 天命：樊嘯天の理性……**(上限99、＋加値：樊嘯天好感度)：**🧾【＜70】なかった**。殺される。
