## 条件と進め方

<b>条件：</b>崆峒派に留学<br>
<b>進め方：</b><br>

### １、郁竹を慕う人に殺される

1. 2年目2月初旬の唐門会議で、**📜 議題：(イベント)崆峒派への留学に行きたい**を選択。
2. **📜 議題：(移動)鉄拳巷**で、2回目の選択で**📖 選択：割り込む**を選択。**🆚 決闘：奪魂弟子**で敗北すると殺される。

### ２、虞小梅を慕う人に殺される

1. 2年目2月初旬の唐門会議で、**📜 議題：(イベント)崆峒派への留学に行きたい**を選択。
2. 2年目3月中旬の行動前に、<Girl3Icon>虞小梅</Girl3Icon>に導かれて崆峒派を見学。<Girl3Icon>虞小梅</Girl3Icon>と知り合う。
3. 加入した崆峒派に応じた対応議題を実行して(自由)修練を行うと、ランダムに<Girl3Icon>虞小梅</Girl3Icon>がジョギングに誘う。**📖 選択：追いかけて行く**。
    - ジョギングへの招待確率は<Girl3Icon>虞小梅</Girl3Icon>の親密度に比例する。
4. ジョギング中にランダムに**🆚 決闘：飛天弟子**が発生。敗北すると殺される。
