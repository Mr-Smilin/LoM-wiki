## 条件と流程

<b>条件：</b>崆峒派に留学<br>
<b>流程：</b><br>

### １、雪山心法の修練で魔が入る

1. 2年目2月初旬の唐門会議で、**📜 議題：(イベント)崆峒派への留学に行きたい**を選択。
2. 2年目3月初旬に門派抽籤が必要で、奪魂門に入った時に<Girl5Icon>夏侯蘭</Girl5Icon>を知ることができる
    - 2回目の抉擇で**📖 抉擇：師門の名誉のため同意する**。
    - 2回目の抉擇で**📖 抉擇：別派に敵視される覚悟を持って反対する。**→**📖 抉擇：構わない。一緒に抽籤しよう。魏掌門に迷惑をかけないために**。
3. **🎲 天命：門派の抽籤結果……**(上限99)：**🧾【＜75】奪魂門**を骰子で出す。<span style='color: Yellow;'>**奪魂門に加入**</span>。
4. 2年目4月初旬の行動前に奇襲を受ける：
    1. **🎲 天命：結果……**(上限90、＋加値：拳掌)：**🧾【＞50】防御成功**。
    2. **🆚 決闘：峨嵋弟子**で勝利が必要 →**🆚 決闘：峨嵋弟子(&玄宮弟子《臨陣指導》)**で敗北するか、4回合終了で戦闘を終える。
    3. 森の中に逃げた後、🧾議題を任意に選択。森から出た後、**🆚 決闘：峨嵋弟子(&玄宮弟子《臨陣指導》)**で勝負に関係なく戦闘を終える。
    4. <Girl5Icon>夏侯蘭</Girl5Icon>が登場。2回目の抉擇で**📖 抉擇：師弟の礼で感謝する**を選択。<span style='color: Yellow;'>**夏侯蘭に拜師**</span>。
5. **📜 議題：(移動)奪魂森林**で、累積2回ランダムに師傅を見つめることで心が散漫になり、師傅を怒らせる(<Girl5Icon>夏侯蘭</Girl5Icon>好感度-1)→**📖 抉擇：師父が素敵だから**。
6. **📜 議題：(移動)奪魂森林**：
    1. <Girl5Icon>夏侯蘭</Girl5Icon>好感度の分岐：
        - <Girl5Icon>夏侯蘭</Girl5Icon>好感度＜35：**📖 抉擇：嫌だ！**：➡️2.<Girl5Icon>夏侯蘭</Girl5Icon>の提案。
        - <Girl5Icon>夏侯蘭</Girl5Icon>好感度≧35：➡️2.<Girl5Icon>夏侯蘭</Girl5Icon>の提案。
    2. <Girl5Icon>夏侯蘭</Girl5Icon>の提案：**📖 抉擇：わかった**
    3. **🎲 天命：修練の成果……**(上限99、＋加値：修養・心相、－減値：内功)：**🧾【＜20】魔が入った**。自爆して死ぬ。

### ２、魏菊による気功治療で爆殺される

1. 2年目2月初旬の唐門会議で、**📜 議題：(イベント)崆峒派への留学に行きたい**を選択。
2. 2年目10月初旬の江陵囲城での丐幫決戦。王二壯が挑戦を宣言する時に、
    1. **🎲 天命：挑戦者は……**(上限99)：**🧾【≦80】ヤバい人物が出てきた**。
    2. 心繋を満たす：<Girl7Icon>魏菊</Girl7Icon>が代わりに掌を受ける。
    3. **📖 抉擇：わかった**。
    4. **🎲 天命：気功で傷を癒す……**(上限40、＋加値：内功)：**🧾【＜80】気功で爆殺される**。爆発で殺される。
