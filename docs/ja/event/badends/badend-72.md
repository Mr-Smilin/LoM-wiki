## 条件と流れ

<b>条件：</b><Girl5Icon>夏侯蘭</Girl5Icon>に師事し、雪山行を発動させる<br>
<b>流れ：</b><br>

1. 2年目2月初旬の唐門会議で、**📜 議題：(イベント)崆峒派への留学に行きたい**を選択。
2. 2年目3月初旬に門派抽籤が必要で、奪魂門に入った時に<Girl5Icon>夏侯蘭</Girl5Icon>を知ることができる
    - 2回目の抉擇で**📖 抉擇：師門の名誉のため同意する**。
    - 2回目の抉擇で**📖 抉擇：別派に敵視される覚悟を持って反対する。**→**📖 抉擇：構わない。一緒に抽籤しよう。魏掌門に迷惑をかけないために**。
3. **🎲 天命：門派の抽籤結果……**(上限99)：**🧾【＜75】奪魂門**を骰子で出す。<span style='color: Yellow;'><strong>奪魂門に加入</strong></span>。
4. 2年目4月初旬の行動前に奇襲を受ける：
    1. **🎲 天命：結果……**(上限90、＋加値：拳掌)：**🧾【＞50】防御成功**。
    2. **🆚 決闘：峨嵋弟子**で勝利が必要 →**🆚 決闘：峨嵋弟子(&玄宮弟子《臨陣指導》)**で敗北するか、4ターン終了で戦闘を終える。
    3. 森の中に逃げた後、🧾議題を任意に選択。森から出た後、**🆚 決闘：峨嵋弟子(&玄宮弟子《臨陣指導》)**で勝負に関係なく戦闘を終える。
    4. <Girl5Icon>夏侯蘭</Girl5Icon>が登場。2回目の抉擇で**📖 抉擇：師弟の礼で感謝する**を選択。<span style='color: Yellow;'><strong>夏侯蘭に師事する</strong></span>。
5. 3年目6月初旬の唐門会議で、**📜 議題：(イベント)雪山行**。
6. 3年目6月下旬、**🎲 天命：お前たちのお昼ご飯……**(上限99)：**🧾【≧66】野生の豚が襲ってきた！**。**🆚 決闘：野豚**で敗北するとぶつかられて死ぬ。
