# {{ $frontmatter.title }}

今から二十数年前、崆峒の[火龍仙君](/ja/people/characters/special204)、全真の[丘處機](/ja/people/characters/special209)、[郝大通](/ja/people/characters/other4)、青城の[鄒博](/ja/people/characters/special201)、[趙逵](/ja/people/characters/special403)が一堂に会し、経を談じ、道を論じたことがあった。
<br><br>
この辯論会では、鄒掌門と郝道長は意気投合し、一見して知り合ったかのようだった。鄒掌門は符籙法術を郝真人に伝授し、郝道長は煉丹の妙技を報いた。
<br><br>
火龍真君は道書を曲解し、意味を勝手に解釈し、道家の学問を仮装して自分を誇耀していた。趙逵も幾分かその影響を受け、後に心が沈落して地獄道法王に堕落することになった。鄒博は歯がみしするほど憎らしかったが、全真の丘真人と南宮の老太爺が言いなだめてくれなければ、彼の若き日の激しい気質で、とっくに崆峒に殴り込んで人を殺していたに違いない。
