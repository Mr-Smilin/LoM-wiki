# {{ $frontmatter.title }}

約二十年前、西域の[極樂教](/ja/people/factions/elysium-school)が武林に大きな脅威をもたらし、勢力を急速に拡大していた。大々的に兵を募り、西夏と内応して河西回廊を奪い、南北の交通路を断ち、大宋と金国の咽喉を握って、中原を奴隷化する企てを持っていた。

<br>

[嵩山派](/ja/people/factions/mount-song-sect)の高僧は衆生の苦しみを憐れみ、身を捨てて正義を貫いた。その行動に感銘した武林の各派の首領は、かつての対立を捨てて、新任の武林盟主[龍淵](/ja/people/characters/special2)の指導の下、心を一つにして[極樂教](/ja/people/factions/elysium-school)を討伐した。

<br>

万仙陣は[嵩山派](/ja/people/factions/mount-song-sect)と[唐門](/ja/people/factions/tang-men)が力を合わせて破壊し、極樂七仙は全員討ち取られた。戦闘の最終段階では、錦香宮の[温夫人](/ja/people/characters/special402)が策を献じ、西夏の援軍を断つことで、極樂教の首領を華山に追い詰め、やむなく正派連盟と決死戦を行わせることになった。

- [唐門](/ja/people/factions/tang-men)の[唐中翎](/ja/people/characters/master)は毒薬を飲んで力を強引に高め、飛燕流星翎で右護法を討ち取った。ただしこれ以来、後遺症が残ることになる。
- [左護法](/ja/people/characters/special213)は青城の[趙逵](/ja/people/characters/special403)を打ち破ったが、逃げおおせた。多年後に再び現れたが、心が崩れ、心から魔が生まれ、今日の[泥教](/ja/people/factions/mud-school)の地獄道法王となってしまった。
- 盟主[龍淵](/ja/people/characters/special2)は天觴剣を握り、死を覚悟して《天罡伏魔剣》で鬼蓮王刀を破り、万悪の首魁「[快活王](/ja/people/characters/other7)」を誅殺した。龍淵の行方は不明となる。

この決戦は、目撃者は稀であり、結末はなおさら哀切であった。[唐門](/ja/people/factions/tang-men)と[嵩山派](/ja/people/factions/mount-song-sect)の両派は元気を失ったが、嵩山はなお世人から尊敬されるのに対し、[唐門](/ja/people/factions/tang-men)は皆から叩かれ続ける犬扱いされるようになってしまったのだ。

## 翼王の背信

この戦いで死傷者は多く、[極樂教](/ja/people/factions/elysium-school)は滅亡したが、正道も大打撃を受けた。ただし[龍淵](/ja/people/characters/special2)の計画では、このようなことにはならないはずだった。

<br>

[龍淵](/ja/people/characters/special2)の義兄弟である金国皇帝、翼王[完顏珣](/ja/people/characters/special819)は、当初は援軍を派遣して「趙を囲んで曹を救う」作戦を約束していた。だが最終的には岸から眺めるだけに決め、武林人士と[極樂教](/ja/people/factions/elysium-school)が激しく戦う間に、漁夫の利を得ようとし、後日また宋国に兵を侵して騒ぎを起こした。

<br>

[龍淵](/ja/people/characters/special2)は戦後、[翼王](/ja/people/characters/special819)の不義を責めるために訪ねたが、翼王は不義ながらも、龍淵は昔日の情義を忘れず、結局手を下すことはできなかった。そこで自分の右腕を切り落とすことで、彼の罪を許すことにしたのだ。中原の豪傑たちに顔向けできず、また娘の[龍湘](/ja/people/characters/girl8)が巻き込まれることを懸念し、身分を隠して[丐幫](/ja/people/factions/beggar-gang)の中に紛れ込むことにしたのだ。

## 備考

- 当初は三対三ではなく、五対五だったという説もあり、例えば[唐中翎](/ja/people/characters/master)の列伝では正派代表が五人いることが述べられている。もしかして初期設定では左右使と左右護法が四人だったのかもしれず、鳥熊はこれについてノーコメントとしている。[^1]
- 左右使は一部のテキストで矛盾があったが、v1.0.3202で左使は逃亡、右使は[唐中翎](/ja/people/characters/master)に殺されたと統一された。
- [左護法](/ja/people/characters/special213)の後の活動は、朝野を震撼させた「[白風峒変乱](/ja/event/stories/白風峒變亂)」という事件を引き起こし、最終的に[唐布衣](/ja/people/characters/brother1)、[解無塵](/ja/people/characters/special808)、[夏侯蘭](/ja/people/characters/girl5)及び他の二人によって討ち取られた。
- 右護法は奇毒の《屍心蟲》を使い、正道の武林人士を支配していた。[唐中翎](/ja/people/characters/master)が無理をして出手した理由は、『屍心丹』に中毒していた[師妹](/ja/people/characters/other10)を救うためだったと思われる。
- 巴哈によるこの戦役の地理的考察：[リンク](https://forum.gamer.com.tw/C.php?bsn=73317&snA=1778&tnum=4&bPage=2)

[^1]: PTT C洽－[\[活俠\] 龍湘と奇怪な小知識P3](https://www.ptt.cc/bbs/C_Chat/M.1729093866.A.C8A.html)
