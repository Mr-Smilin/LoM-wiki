## 詳細な流れ

:::info
本セクションは体験版 Lv2 時点の掲示板解説 ([巴哈姆特・講經樓](https://forum.gamer.com.tw/Co.php?bsn=73317&sn=259)) を日本語版 wiki が翻訳したものです。製品版とは細部が異なる可能性があります。
:::

**毎月の唐門月例会（評点後は1日休暇）**において、外姓弟子は最資深の同門弟子から月の貢献を評価される。

1. **評点官による貢献評価**
   + 評点官が [趙活](/ja/people/characters/player) を集合させ、月の貢献度から **50を減算**
   + 通常、貢献度 50 が基準ライン

2. **🧾【≧50】達成：月間貢献度 50 以上**
   + <span style='color: #00FF7F;'>門中の上下で奔走し、多くの雑務をまとめる</span>
   + 評点官より感謝の言葉 → **心相+15**、**向心提升**
   + 同門の面前で酒宴に招かれ、談笑

   ::: details 会話全文（達成時）
   **評点官（通常）**：師弟よ、苦労をかけたな。このところ門中は外姓弟子がおまえ一人だというのに、これほどまでに多くの雑務をまとめてくれるとは。皆、日ごろは口にしないが、心底では大変感謝しておる。

   **趙活**：ご誉めいただき、ありがとうございます。

   **評点官（開心）**：次の段階試験に向けて頑張れよ。そうして、おまえも入室して一緒に功を練ることができれば、我が門派の名も高まるというもの。ただ、そうなるとしたら、雑務をやる者がいなくなるな。ハハハ！

   **評点官（開心）**：そんなことは気にするな、橋の上に来れば自ずと川を渡るというもの。毎度毎度同じせりふばかりでは飽きもする。さあ、皆で一緒に酒を飲んで、ちょっと賭け事でもしよう。

   **趙活**：共敬の心は従うことにしかず。
   :::

3. **🧾【＜50】未達成：月間貢献度 50 未満**
   + 評点官が不足を指摘 → **📖抉擇：対応を選択**

::: details 会話全文（未達成時の指摘）
**評点官（通常）**：師弟よ、これからは頑張らねばならんぞ。

**評点官（通常）**：別に我が人情知らずというわけではない。いまも外姓弟子がおまえ一人だし、雑務を皆おまえの上に押しつけるのは確かに不公平だろう。しかし門規がそう定めているので、誰もが同じように通ってきたのだ。

**評点官（通常）**：おまえは外姓弟子である以上、自分の本分をわきまえねばならん。何かにつけて手を抜き、怠けてばかりいる。ただ自分のことばかり考えている。皆がおまえのようにただ身の周りのことばかり気にかけていたら、天下は大混乱ではないか？
:::

   + **📖抉擇：屁理屈をこねる**
     - 口先で師兄を説き伏せ、告げ口を回避
     - **道徳+1、処世-2、弁舌+1**
     - その後、同門が弟子房で悪口を言うのを聞く
     - **心相-20、向心下降、掌門好感度下降**

   ::: details 会話全文（屁理屈選択時）
   **趙活**：待ってください、師兄。理由があるんです。聞いてください——

   **評点官（通常）**：まあいい。今回は許してやる。ただし、ちゃんと頑張っていないと、次の月には必ず掌門にご報告し、御老人の判断をあおぐからな。

   （その後、二人の唐門弟子が弟子房で愚痴をこぼす）

   **唐門弟子甲（通常）**：今日はようやく忙しさがおわった。水を汲んだり薪を割ったり、本当に疲れたぞ。結局のところ、おれは唐門に功を練りに来たのか、雑務をしに来たのか。これは外姓弟子がやるべき仕事ではないか。

   **唐門弟子乙（通常）**：今は入室弟子が外姓弟子より多いんだ。我ら二人が手を出さなきゃ誰がやるんだ？

   **唐門弟子甲（通常）**：金を持ってきて孝敬する師兄たちは、我ら二人よりずっと運がいいんだ。資歴は大して変わらんのに、雑務が少なくてすむんだぞ。

   **唐門弟子乙（通常）**：愚痴を言うなよ。人それぞれの運命というやつだ。金を出す者もいれば、力を出す者もいる。それまでだ。金持ちの奴らの援助がなきゃ、我らも飯にはありつけん。おれたちみたいな貧乏人が拝師に来たんだから、どうしても他人より苦労が多くなるもんだ。

   **唐門弟子甲（通常）**：そりゃそうだな。おれだって疲れた時についこぼやいちゃったんだ。本当に仕事をしないってわけじゃなきゃ、心の中もすっきりしないしな。

   **唐門弟子乙（通常）**：そういうもんだ。

   **唐門弟子甲（通常）**：そういえば、金も出さんし力も出さん奴がいるぜ。見てもむしゃくしゃするほどだ。人が忙しくしてる時、奴ときたら身を隠しちゃって、何をしてるのやら知ったこっちゃない。とにかく本業をしてやがらん。おれたちが目をつぶってると思ってんのか。

   **唐門弟子乙（通常）**：心配するな。師父や師兄たちの眼は節穴じゃねえ。だからこそ、この野郎は外姓弟子のまんまで、我らだけが入室できたんだよ。

   **唐門弟子甲（通常）**：そりゃそうだ。さあ、天がまだ暗くならないうちに、稽古場に行ってついてこい。体をほぐすために何手か試してみようぜ。
   :::

   + **📖抉擇：黙る**
     - 無言で非難を受け入れる
     - **性情-1、唐中翎-2**
     - 掌門に全く報告されることが決定
     - その後、同門が弟子房で悪口を言う（屁理屈選択と同じ流れ）

   ::: details 会話全文（黙る選択時）
   （おまえは無言で非難を受け、師兄は見て分かるように、おまえが弁明しないのを見て、突っ込む気も失った）

   **評点官（通常）**：師弟よ、これが不公平だというのはわかっているが、ただおまえの寛容を頼むしかない。もちろん、おまえが拜師入室の気もなければ、そっとしておくのもいい。

   **評点官（通常）**：ともあれ、今日のことは、ありのままに掌門にご報告する。いいな？

   **趙活**：かしこまりました。
   :::

   + **📖抉擇：逆ギレ** （道徳 < 40 で触発可能）
     - 評点官と同門が激怒
     - **🆚戦闘：同門が怒る**
     - 戦闘結果に関わらず、掌門が <span style='color: red;'>師門から逐出</span>
     - <span style='color: red;'>年月経過後、丐幫に身を落とし、唐門への恨みを募らせる</span>
     - <span style='color: red;'>夜間襲撃 → 小師妹の暗器で撃ち殺される</span>
     - [生死簿 No.10：小師妹の制裁](/ja/event/badends/#生死簿-No.10)

   ::: details 会話全文（逆ギレ選択時）
   **評点官（生气）**：馬鹿も休み休み言え！もちろんお前に決まってるだろ！おまえがやらん、やり切れんなら、黙ってそう言え。誰も後ろ手に縛って無理やりさせたりしねえ。その気がなきゃ、正直に言ってさっさと出て行け。

   **評点官（生气）**：おまえは掌門や小師妹の前でどんな大きなことを言ったんだ？いまになって自分のことだけ考えて、事を成し遂げられず、逆に他人を貶すようなことばかり言ってる。どうしてこんなに厚顔なんだ。

   **評点官（生气）**：おまえが思ってるか知らんが、この門派の雑務が本当におまえ一人でやってるもんか。誰だって力の及ぶ限り門派のために心を尽くしてるんだ。おまえのせいで不恥知になるのは嫌だ。

   **唐門弟子甲（生气）**：みんなが見てたぞ、おまえが忙しくてるから、三日と置かずにおまえの分を手伝ってやってたんだ。おまえは本当に知らんのか、それとも薪が自動で割れると思ってるのか、米飯が自動で炊けると思ってるのか？みんなで力を合わせてこそ、この門派が立ってるんだ。おまえ一人の功労だと思ってるのか？

   **唐門弟子乙（生气）**：その馬鹿みてえなことを言うな。派遣された仕事がおまえの時間全部を占めてるのか？入室弟子は雑務は少なくても、毎日下山して金を稼がなきゃならんし、本門の借金を返さなきゃならんし。おまえはここまで、みんなが稼いだ金で生きてるんだぞ！どうしてこんな自分勝手なんだ。

   **唐門弟子丙（生气）**：醜い面なんかで済ますか。そんな不潔なことを言うんじゃあ、おまえの心根のきたなさはそれ以上だな。

   **評点官（生气）**：もう我慢できん。つべこべ言うな、こいつを打ち殺してやる。我を止めるな！

   **唐門弟子甲（生气）**：待ちな。おれがやってやる！世人が誅するに値する此の糞野郎は、おれが懲らしめてやる！

   **唐門弟子乙（生气）**：おれも手伝うぜ、師兄！
   :::
