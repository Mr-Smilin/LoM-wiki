			招待状を届ける。江陵包囲戦以来の江湖の乱れを受け、皇帝の聖旨により上官家当主が
			使者に任じられ、朝野の関係を修復し武林大会を開催して武林盟を再び結ぶ、という
			朝廷主導の大会で、各門派には官員が同行する。会場は風雨山。<br>
			<br>
			「ゆえに官家は特例として天恩を垂れ、聖旨を下されました。」<br>

			主人公・趙活が唐門一行の隊長に推され、「無理です(正直)」「できます(嘘)」の
			選択肢が出る。
