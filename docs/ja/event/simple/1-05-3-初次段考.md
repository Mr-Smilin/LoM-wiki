1. **🎲天命：段階試験の考官……**(上限99)
   + **🧾【>90】掌門人**：[唐中翎](/ja/people/characters/master)の登場に弟子たちの士気が高まる
   + **🧾【>50】三師兄**：[唐陞](/ja/people/characters/brother3)が考官を代行する
   + **🧾【>20】二師兄**：[唐錚](/ja/people/characters/brother2)、練武場に異様な緊張感が漂う
   + **🧾【≦20】大師兄**：[唐布衣](/ja/people/characters/brother1)が試験内容の変更を試みて三師兄に制止され、全員の戦意が失せる
2. **🆚戦**：唐門弟子Bとの一対一の対戦 (秘笈イベントの既出/未経験で戦闘前の台詞が変化)
   + <span style='color: #00FF7F;'>勝利：**武学上昇、心相+30、向心低下** → 考官別の講評、夜パートへ</span>

### 考官分岐 — 勝利の講評

勝利すると考官・<Girl0Icon>小師妹</Girl0Icon>・四師兄が信じられない目で見守り、周囲の議論が沸き立つ (秘笈イベント既出の場合は唐門弟子ABCが周囲に笑われて去る一幕あり)。

1. **考官が掌門人の場合**：趙活の悟性の高さを評価。**掌門人好感度上昇**。<span style='color: red;'>直後に掌門が吐血して倒れ、その夜、荷造りをして黙って唐門を去る弟子が続出する</span>
2. **考官が三師兄の場合**：無師自通の成長に驚嘆し、掌門への正式入室の推薦を約束する
3. **考官が二師兄の場合**：**二師兄好感度上昇、向心低下**。本門弟子の素質の低さへの叱責が全弟子を総毛立たせる
4. **考官が大師兄の場合**：**📖抉擇：勝利の弁を述べるか**
   + **📖抉擇：述べる**：**性情上昇、処世上昇、心相+15**、大師兄 (秘笈未経験時は小師妹も) の**好感度上昇**。《甘蔗》を獲得
   + **📖抉擇：述べない**：**修養上昇、処世低下、大師兄好感度上昇**。《甘蔗》を獲得

### 夜パート

1. **男弟子房**：同室の入室弟子たちが趙活を称える
   + **秘笈イベント既出の場合**：唐門弟子ABC三人が乱入し、追い返される → **📖抉擇：**
     + **📖抉擇：爽快だ!**：**処世上昇、心相+15**
     + **📖抉擇：そんなことないよ、和を大事にした方がいいだろ**：同門の絆を大事にする
2. **男弟子房外**：<Girl0Icon>小師妹</Girl0Icon>が訪れ、掌門の体調不良で入室が見送りになったと伝える (**心相-10 が3回**)。<Girl0Icon>小師妹</Girl0Icon>が趙活の背をそっと叩いて慰める (**心相+30**) → **📖抉擇：**
   + **📖抉擇：まだちょっと悲しい、もっと叩いてもらう**：**心相+30、小師妹好感度上昇**
   + **📖抉擇：悲しくない**：**心相+10、小師妹好感度上昇**
   + **📖抉擇：何の役に立つんだ!**：**処世上昇、修養低下、小師妹好感度低下**

### 敗北の場合

+ **性情低下、心相-20**
+ **秘笈イベント既出の場合**：<span style='color: red;'>趙活は暴言を吐いて唐門を出奔。多年後、唐門陣営への夜襲の末に<Girl0Icon>小師妹</Girl0Icon>の暗器に斃れる → [生死簿 No.10](/ja/event/badends/#生死簿-No.10)「小師妹の制裁」へ進む</span>
+ **秘笈イベント未経験の場合**：唐門弟子Bに慰められ、静かに終わる
