## 基本の流れ

**カテゴリ**：メインシナリオ(青城送りの道中)<br>
**発生地**：唐門 → 青城派への道中<br>
**発生時期**：第2年8月下旬頃 ([廢寺喬裝](/ja/event/detailed_description/2-08-3-廢寺喬裝)直後、自由行動を挟まず連続発生)<br>
**前提**：[廢寺喬裝](/ja/event/detailed_description/2-08-3-廢寺喬裝) 分岐Bで主人公が負傷し青城派へ搬送される道中 → 後続は ➡️ [青城での療養](/ja/event/detailed_description/2-08-5-青城療傷) (TODO: 各場面の発生地・接続順は再検証中)<br>

1. **📜前兆：段智秀来襲の報**：青天の霹靂(落雷か)を「災いの前兆」と読み、[葉雲裳](/ja/people/characters/girl2)・[葉雲舟](/ja/people/characters/special3) が動揺し荷造りを始める
   + **📖選択：留まる**：「段智秀が現れたってことは、とっくに包囲網が敷かれてるはず」「鄒掌門が山にいる限り、あいつも悪さを働くには多少の遠慮がいる」と判断する → ➡️2.
2. **📜対峙：段智秀との遭遇**：葉雲舟が[段智秀](/ja/people/characters/special833)に殴りかかるが「私が片手で相手をしてやっても、君は本公子の敵ではない」と一蹴される (TODO: この対峙が実際の戦闘に発展するか、台詞のみの場面かは未確認)
   + 葉雲裳が「あいつの内功は深いから、攻撃しても逆に自分が傷つくだけ」と兄を止める → ➡️3.
3. **📜襲撃：青城襲撃**：「拝謁の書状を届けに来た」と称する一団(実態は襲撃者)が青城を襲撃する。傷が治ったばかりの主人公は「趙兄弟、この連中は青城を狙って来たんだ…命を無駄にするな、早く逃げろ！」と忠告される
   + **📖選択：残留・参戦する**：「恩知らずとして生きるのは生きてるうちに入らない」「今日が俺の命日なら、遺骨を火葬して唐門に送ってくれ」 → ➡️4.
   + **📖選択：忠告に従い早期離脱する** (TODO: 選択した場合の分岐内容は未確認。離脱後に4.へ合流するか別展開かも不明)
4. **📜脱出行：役割分担での脱出**：「師姉、雲裳と御者を守ってください…万一に備えて地面に罠を仕掛けます」/ [唐芳](/ja/people/characters/special832)「意外と師弟は頼りになるな」。道中「唐門での日々がどれだけクソみてえなもんか」と本音を語り合う場面を経て、唐門へ帰還する描写がある (TODO: 戦わずに逃げた後、改めて青城へ向かうのか、この帰還が後続の別イベントの観測混入なのか、[青城での療養](/ja/event/detailed_description/2-08-5-青城療傷)との接続順は未確認) → イベント終了
