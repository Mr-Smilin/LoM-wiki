# {{ $frontmatter.title }}

**カテゴリ**：イベント

## 存活の鍵

第3年1月中旬、**🎲天命：幾人の援手が必要か？....**(上限60、➕加値：[唐布衣](/ja/people/characters/brother1)好感度）：
   + **🧾【＜80】必要ない**：[唐布衣](/ja/people/characters/brother1)と一緒に下山しない．
   + **🧾【≧80】必要ない、俺と趙活で十分だ。**、後で以下を選ぶ：
      + **📖抉擇：嫌だよ(猛烈に抵抗)。**：[唐布衣](/ja/people/characters/brother1)と一緒に下山しない．
      + **📖抉擇：仕方ねえな**：[唐布衣](/ja/people/characters/brother1)と一緒に下山する．

### 唐門に待機

第3年1月下旬、[唐布衣](/ja/people/characters/brother1)は重傷で帰ってくる。
   1. 医術スキル≧5 または 総煉丹消費ポイント≥250、[唐布衣](/ja/people/characters/brother1)の治療を決定する．
      + **📖抉擇：躊躇** は結果に影響しない、3回後は自動的に決意する．
   2. 治療開始、内力-3~-5の後、以下の条件を検定、分数≧2であれば、大師兄は生存．
      + 内力≧25、加1点．
      + 内力≧50、加1点．
      + スキル：南宮護心訣≧3、加1点．
      + 途中で心を鎮める（小梅不在）、-1点．

補足事項：
+ 「内力削除後」に検定する、最低でも内力55を保つか、[南宮心法](/ja/system/books/book_7012)の南宮護心訣を習得することをお勧めする．
+ 虞小梅が唐門に加入した場合、「心を鎮める」を選ぶと孤島エンディングに、「鎮めない」を選ぶと敵を撃退する手助けをする．

### 一緒に下山

1. 第3年1月下旬 唐門に戻る途中で[金烏上人](/ja/people/characters/special207)に奇襲される、何らかの対象と戦う．
2. [金烏上人](/ja/people/characters/special207)に頭を叩かれず、内力を吸収される時、以下の条件のいずれかを満たすと大師兄は生存：
   + 名聲≧50かつ道德≧80【俠客】．
   + [申屠龍](/ja/people/characters/special405)を知っており好感≥30．
3. 頭を叩かれたか上記の条件を満たさない場合、命運≧10が必要、**📖抉擇：どうしても大師兄には生きていてほしい**を選ぶ．

## 帰還の時期

イベント当初は大師兄が生存しているかどうか分からず、以下のタイミングでようやく帰ってくる：

+ **西武林盟ストーリーライン**：第4年2月中旬、[東西武林盟會戰](/ja/event/detailed_description/4-02-2-東西武林盟會戰)、結緣対象がいなくても可．
+ **頑抗到底ストーリーライン**：第4年1月中旬、[唐門圍攻戰](/ja/event/detailed_description/4-01-2-唐門圍攻戰)、結緣対象がなく、[瑞笙](/ja/people/characters/special999)武林盟主に勝利して[汗青書46：相聲二人組](/ja/event/ends/end-46)に進む．

## 備考

- 下山で命運を消費するのが最も簡単な方法である．
- 小梅が唐門に加入：
   1. 第2年2月上旬唐門会議で**📜議題：崆峒派に留学したい**を選ぶ．
   2. 第2年10月下旬、虞小梅好感度≧20．
- 申屠龍を知る：
   1. 葉家兄妹を受け入れる．
   2. 留学しない、脱出後に廃寺に住む．
   3. 唐門防守、心上人が[葉雲裳](/ja/people/characters/girl2)、青城留学が発生する．
