# {{ $frontmatter.title }}

**カテゴリ**：イベント

## 存活の鍵

第3年1月中旬、**🎲天命：幾人の援手が必要か？....**(上限60、➕加値：[唐布衣](/ja/people/characters/brother1)好感度）：
   + **🧾【＜80】必要ない**：[唐布衣](/ja/people/characters/brother1)と一緒に下山しない。
   + **🧾【≧80】必要ない、俺と趙活で十分だ。**、その後以下を選ぶ：
      + **📖選択：嫌だよ(猛烈に抵抗)。**：[唐布衣](/ja/people/characters/brother1)と一緒に下山しない。
      + **📖選択：仕方ねえな**：[唐布衣](/ja/people/characters/brother1)と一緒に下山する。

### 唐門に待機

第3年1月下旬、[唐布衣](/ja/people/characters/brother1)は重傷で帰ってくる。
   1. 医術スキル≧5 または 総煉丹消費ポイント≥250、[唐布衣](/ja/people/characters/brother1)の治療に当たる。
      + **📖選択：躊躇** は結果に影響しないため、3回後は自動的に決意される。
   2. 治療開始、内力-3~-5の後、以下の条件を検定して、分数≧2であれば大師兄は生き残る。
      + 内力≧25、加1点。
      + 内力≧50、加1点。
      + スキル：南宮護心訣≧3、加1点。
      + 途中で心を鎮める（小梅不在）、-1点。

補足事項：
+ 「内力削除後」に検定するため、最低でも内力55を保つか、[南宮心法](/ja/system/books/book_7012)の南宮護心訣を習得することをお勧めします。
+ <Girl3Icon>虞小梅</Girl3Icon>が唐門に加入した場合、「心を鎮める」を選ぶと孤島エンディングに進み、「鎮めない」を選ぶと敵を撃退するのに協力してくれる。

::: details 虞小梅が唐門に加入する条件
1. 第2年2月上旬唐門会議で**📜議題：崆峒派に留学したい**を選ぶ。
2. 第2年10月下旬、<Girl3Icon>虞小梅</Girl3Icon>好感度≧20。
:::

### 一緒に下山

1. 第3年1月下旬 唐門に戻る途中で[金烏上人](/ja/people/characters/special207)に奇襲される、何らかの対象と戦う。
2. [金烏上人](/ja/people/characters/special207)に頭を叩かれず内力が吸収される場合、以下の条件のいずれかを満たすと大師兄は生き残る：
   + 名声≧50かつ道徳≧80【俠客】。
   + [申屠龍](/ja/people/characters/special405)を知っており好感≥30。
3. 頭を叩かれたか上記の条件を満たさない場合、命運≧10(下山で消費するのが最も簡単な入手方法)が必要、**📖選択：どうしても大師兄には生きていてほしい**を選ぶ。

::: details 申屠龍を知る条件
1. 葉家兄妹を受け入れる。
2. 留学しない、脱出後に廃寺に住む。
3. 唐門防守、想い人が[葉雲裳](/ja/people/characters/girl2)、青城留学が発生する。
:::

## 帰還の時期

+ **前提**：イベント当初は大師兄の生死が不明のままで、以下のいずれかのタイミングで帰還する
+ **西武林盟ストーリーライン**：第4年2月中旬、[東西武林盟會戰](/ja/event/detailed_description/4-02-2-東西武林盟會戰)、結緣対象がいなくても可。
+ **頑抗到底ストーリーライン**：第4年1月中旬、[唐門圍攻戰](/ja/event/detailed_description/4-01-2-唐門圍攻戰)、結緣対象がなく、[武林盟主](/ja/people/characters/special999)に勝利して<EndIcon no="46">汗青書46：漫才二人組</EndIcon>に進む。
