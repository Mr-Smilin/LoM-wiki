## 基本流程
**カテゴリ**：メインシナリオ<br>
**発生時期**：第3年8月上旬 唐門月例会<br>
**発動条件**：白天に正心堂へ行くと自動開始 (カレンダー固定)<br>

1. **唐門会議：**[四師兄](/ja/people/characters/brother4)が「旅商隊を組んで山を下り、蜀の特産品を売りながらゆくゆくは金国・大漠まで行く」と離門を宣言 (掌門の許可は取得済みの事後報告)
   + **📜議題：退席**：[四師兄](/ja/people/characters/brother4)<span style='color: red;'>は唐門を永久離脱する</span>
   + **📜議題：四師兄の遠遊**：
     + **📖選択：その通り** →**📜議題：退席**：[四師兄](/ja/people/characters/brother4)<span style='color: red;'>は唐門を永久離脱する</span>．
     + **📖選択：あえて難題を突きつける**、**🎲天命：四師兄のお返し......**(上限70、➕加値：[唐惟元](/ja/people/characters/brother4)好感度)：
       + **🧾【＜70】感謝還元大セール**：[四師兄](/ja/people/characters/brother4)の店をオープン(商品半額)→**📜議題：退席**：[四師兄](/ja/people/characters/brother4)<span style='color: red;'>は唐門を永久離脱する</span>．
       + **🧾【≧70】さあ、地元の土産品だな**：
         + **📖選択：小師妹がどう過ごしているか知りたい**(<Girl0Icon>小師妹</Girl0Icon>と結ばれていない場合のみ→**📜議題：退席**：[四師兄](/ja/people/characters/brother4)<span style='color: #00FF7F;'>は第3年11月上旬に最後の行動時点で唐門に戻る</span>．
         + **📖選択：華山には仙人がいると聞いた、仙人は霊丹を持つ**→**📜議題：退席**：[四師兄](/ja/people/characters/brother4)<span style='color: #00FF7F;'>は第3年11月上旬に最後の行動時点で唐門に戻る</span>．
2. **別宴：**その夜、送別の宴。[四師兄](/ja/people/characters/brother4)が[唐陞](/ja/people/characters/brother3)へ手間のかかる料理(鵪の飾り蒸し、海老豚足膾、五珍膾など)を注文し、[小師妹](/ja/people/characters/girl0)も同席する(「飲めるけど、お父さんがダメって。」)
   + 料理スキル≧40：[四師兄](/ja/people/characters/brother4)の注文に即快諾する．
   + 料理スキル≧30：できる限り試してみるが、本格的な仕上がりは保証しない．
   + 料理スキル≧20：[四師兄](/ja/people/characters/brother4)の注文は聞いたことあるが、自分では作れない．
   + 料理スキル＜20：[四師兄](/ja/people/characters/brother4)の注文は全く理解できない．
3. ➡️ 次は[武林大会の招待状](/ja/event/simple/3-08-3-武林大會請柬)(8月下旬)。土産を頼んだ場合の帰還は[四師兄が帰る](/ja/event/detailed_description/3-11-1-四師兄歸來)を参照
