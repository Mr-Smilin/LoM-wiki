## 基本の流れ

**カテゴリ**：メインシナリオ(想い人＝[葉雲裳](/ja/people/characters/girl2)ルート派生)<br>
**発生地**：青城派・上清宮(道観)<br>
**発生時期**：第2年8月下旬〜9月下旬頃<br>
**発動条件**：[廃寺の変装者たち](/ja/event/detailed_description/2-08-3-廢寺喬裝) 分岐Bで
主人公が重傷を負い、かつ想い人が[葉雲裳](/ja/people/characters/girl2)であること
(TODO: 他の想い人の場合は崆峒派へ送られる別ルートとなり、本ページのイベント群は発生しない。
崆峒ルートの具体的な内容は未収録)<br>
**接続**：[廢寺喬裝](/ja/event/detailed_description/2-08-3-廢寺喬裝) → 本イベント、と自由行動を
挟まず連続発生。本イベントの日常イベント群を一通り経た後、留学の終盤に
[段智秀來襲](/ja/event/simple/2-09-3-段智秀來襲)が発生し、青城留学が終わる<br>

1. **青城送り：**
   + 掌門は、二師兄・[唐錚](/ja/people/characters/brother2)の医術では内傷の治療に
     「内功を廃す」荒療治しかなく主人公から恨まれ続けるのを避けたいという配慮と、
     青城派の玄門正統の内功が陰陽の気を調和させるのに最適という医療的必然性の両方から、
     唐門と特に親しい六大派の一角である青城派へ主人公を送り、療養と「留学」の名目で
     滞在させる
   + 運んだのは唐門の女医・師姉の[唐芳](/ja/people/characters/special832)。
     [葉雲裳](/ja/people/characters/girl2)も無理を押して同行
   + 青城派側の窓口は前任掌門の直弟子で道号「逸塵道子」と呼ばれる若き道士・
     [申屠龍](/ja/people/characters/special405)(鄒掌門・[鄒博](/ja/people/characters/special201)の代理)
   + 唐門からの手紙で事情を知った申屠龍が自ら運功治療し、主人公は2〜3日で意識を取り戻す
2. **悪夢〜目覚め：**
   + 意識不明の間、主人公は複数の悪夢を見る。いずれも「もし違う選択をしていたら」という
     夢オチの代替シナリオで本編には影響しない(点蒼を出奔して名を上げる想像、申屠龍の脈診を
     拒んで暴走し葉雲裳を死なせてしまう最悪の想像、葉雲舟に「妹を死なせた責任」を問われる
     場面など複数パターン。TODO: どのプレイヤー状態でどの悪夢が選ばれるか
     〈ランダムかステータス依存か〉は未解析)
   + 目覚めると青城派の道観(上清宮)の一室。葉雲裳が真っ先に飛びついて再会を喜び、事情も
     聞かずに部屋を漁る小ネタを挟みつつ、門を出た先で申屠龍と対面する
3. **事情説明：**
   + 唐芳がのんびりと事の顛末を語る：南宮己の一撃を庇った内傷、掌門の応急処置(暗器で
     経穴を突く)、二師兄の鍼治療、そして「二師兄の元にいると恨まれ続けるので青城へ
     送った」という真相
   + 申屠龍が掌門の手紙を読み、自ら運功治療して主人公は数日で回復。青城滞在は「留学」
     扱いとされ、生活・修行は自由
   + ただし葉雲裳は素行不良ゆえ上清宮への立ち入りを制限される(タケノコを座布団に
     詰め込んだ「タケノコ座布団事件」の伏線がここで説明される。詳細は
     [葉雲裳](/ja/people/characters/girl2) ページのトリビア欄を参照)
4. **療養中の唐芳との問診：** ➡️ [唐芳問診](/ja/event/simple/2-08-6-唐芳問診) を参照
5. **「厄介払い」疑惑の否定：**
   + 唐芳問診の全話題消化後、主人公は食事を挟みつつ掌門の推薦状の意図について自問自答する
   + 「掌門は自分が面倒だから青城に押し付けようとしたのではないか」という疑念に対し、
     「掌門はそういう人ではない。邪魔なら、とっくに門から蹴り出しているはずだ」と
     自ら結論づけ、疑惑は否定される(章の締め)
   + ➡️ 続く留学終盤の顛末は [段智秀來襲](/ja/event/simple/2-09-3-段智秀來襲) を参照
