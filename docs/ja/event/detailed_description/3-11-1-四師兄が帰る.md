# {{ $frontmatter.title }}

> 武林大会〜錦香宮滞在 (第三年8〜11月) の一連のメインシナリオの締めくくり。
> 唐門一行の帰路(簡易表の「唐門に帰る」)と、
> [四師兄の旅立ち](/ja/event/simple/3-08-2-四師兄道別)で旅商に出た
> 四師兄・[唐惟元](/ja/people/characters/brother4)の帰還を扱う。
> **注意**: 第二年10月の「[唐門に戻る](/ja/event/simple/2-10-1-返回唐門)」(青城留学からの
> 帰還)とは別のイベント。

## 基本流程

**カテゴリ**：メインシナリオ<br>
**発生時期**：第三年11月 (武林大会閉幕後の帰路〜帰還後)<br>
**発動条件**：メイン進行で自動発生。ただし四師兄の帰還自体は
「四師兄に手土産(土特産)を頼んでいた」ことが前提とみられる(下記TODO)<br>

武林大会が終わり、唐門一行は江陵から唐門への帰路につく。道中の茶店では
「南宮家当主が人間道法王の手にかかり、温夫人のために死んだ」という号外を耳にし、
雨宿りや野営を挟みながら唐門へ帰り着く(「三師兄、ただいま」)。帰還からほどなくして、
旅商に出ていた四師兄も唐門へ戻ってくる。

## 帰路

帰路は独立したイベント扱いではなく、
[錦香宮に泊まる](/ja/event/simple/3-10-1-錦香宮に泊まる)からの連続で進行する(調査済み)。

野営の場面では、武林大会を経て唐門を辞める決意をした古参の外弟子を引き留める選択肢がある:

- 👉「脅す」
- 👉「説得する」
- 👉「引き留めるより、歌でも歌おう」

代表台詞:

- 「君が去れば、唐門の肉が削がれる。」
- 「唐門は家だ。俺たちは師兄弟であると同時に、家族なんだ。」

この選択は後続の分岐で参照される。実プレイの観測では、この帰路〜帰還の
過程で三師兄・[唐陞](/ja/people/characters/brother3)の好感度が上昇した(選択肢との対応は未確認)。

## 四師兄の帰還

主人公の帰還から少し遅れて、四師兄本人が唐門に帰還する。簡易表の
「手土産を頼む必要がある」という注記は、手土産を頼んだかどうかが帰還関連の分岐で
参照されていることと対応するとみられる。

実プレイの観測では、帰還シーンの前後で四師兄の好感度が一度下がってすぐ少し戻る変動があった
(シーン内の選択肢によるものと推測)。
