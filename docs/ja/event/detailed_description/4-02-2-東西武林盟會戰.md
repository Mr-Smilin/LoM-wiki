## 基本流程

**カテゴリ**：メインシナリオ<br>
**発生時期**：第4年2月中旬 残り2回の行動時<br>

### １、西武林大会

1. 西武林盟を支援する門派が到着：
    - 丐幫(新派)：向心-3
    - 丐幫(大義分舵)：増減なし
    - 峨嵋派：向心+2
    - 青城派(西武林盟支援)：<span style='color: Red;'>確認待機</span>
    - 青城派(態度曖昧)：<span style='color: Red;'>確認待機</span>
    - 飛石幫：増減なし
    - [福韞](/ja/people/characters/special206)が唐門に加入：増減なし
    - [解無塵](/ja/people/characters/special808)が唐門に加入：増減なし
    - 崆峒派説得成功：増減なし
2. [南宮深](/ja/people/characters/special102)が山に上って説得：
    - [南宮深](/ja/people/characters/special102)好感度 ≧20：向心-2
    - [南宮深](/ja/people/characters/special102)好感度＜ 20：向心-3
3. 反論：
    - （詳細な分岐条件は複雑なため省略）
4. [南宮深](/ja/people/characters/special102)が退去前に脅迫：向心-2
    - 崆峒派が武林盟を支援：追加向心-1
    - （他の条件も複雑で省略）
5. [南宮深](/ja/people/characters/special102)が山を下りる：
    - **📖 抉擇：来やすいが去るのは難しい！**：嘴力+1、処事+1
    - **📖 抉擇：奴らに行かせよう．**：処事-1、修養+1

### ２、東西武林盟会戦

1. 外堡の疎散：名声-1
2. 衆人を配置：
   - （各キャラの条件に基づいた詳細な配置分岐）
3. 主人公の配置：
    - **📜 議題：(事件)決鬥：三師兄を守る**：[３－１、唐門大院防守ルート](#３－１防守唐門大院路線)へ
    - **📜 議題：(事件)戦役：唐門出撃**：向心+1、[３－２、唐門外堡防守ルート](#３－２防守唐門外堡路線)へ

### ３－１、唐門大院防守ルート

（複雑なイベント分岐のため、主要な流れのみ記載）

1. 同行相手を決定する
2. 刺客との決鬥が発生する
3. （複数の決鬥や天命分岐）
4. 最終的に[瑞笙](/ja/people/characters/special999)との大決戦へ

### ３－２、唐門外堡防守ルート

武林盟の攻撃を防ぎながら、複数の戦役と決鬥が発生する複雑なルート。

### ４、会戦収尾

1. [瑞笙](/ja/people/characters/special999)が撤退を命令
2. 武林盟撤退後：
   - 龍湘を探す
   - 夏侯蘭の傘を修理
   - 小師妹の心配
   - 大師兄の様子を確認（飛俠歸來を満たした場合）
