# {{ $frontmatter.title }}

玄功門の丹青羅刹・閻羅生。かつて飛天門の[丹霞子](/ja/people/characters/special404)、鐵拳門「白眉虎」[孟瞋](/ja/people/characters/other1)、奪魂門の[第三香](/ja/people/characters/special820)と共に、4人で「崆峒四秀」と並称された。判官筆を練り上げ、門派人選挙の掌派人の座を目指す可能性もあった。

<br>

自分の実力が第三香に劣ることを自覚しており、玩物喪志に陥り、書画の中に絶世の武功が隠されていると信じ込んだ。門派人選挙の期日が近づくにつれ、寝食を忘れて書画の研究に没頭した。

<br>

第三香と[呼延菊](/ja/people/characters/special822)の密かな密会を偶然目撃し、その場で激怒して手を出したが、呼延菊が[夏侯蘭](/ja/people/characters/girl5)に向かう判官筆を身を挺して受け止めた。第三香は彼の腸を引き出して殺した。
