# {{ $frontmatter.title }}

全真七子の一人「太古真人」郝大通。老荘と易経に精通し、卜筮占卦の得意な人物。華山に奇遇があると占い、果たして希夷祖師の『睡功』を手に入れた。その後の悟りを通じて、全真華山派を開創した。

<br>

20年以上前に[青城論道](/ja/event/stories/青城論道)に参加し、[鄒博](/ja/people/characters/special201)と相談して大変気が合った。
