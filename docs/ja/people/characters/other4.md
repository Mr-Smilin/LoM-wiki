# {{ $frontmatter.title }}

全真七子の一人「太古真人」郝大通。老荘と易経に精通し、卜筮占卦を得意とする。華山に奇遇ありと占い、果たして希夷祖師の《睡功》を手に入れた。以後悟るところがあり、全真華山派を開いた。
<br><br>
二十数年前に[青城論道](/ja/event/stories/青城論道)へ参加し、[鄒博](/ja/people/characters/special201)と意気投合した。
