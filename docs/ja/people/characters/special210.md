# {{ $frontmatter.title }}

史実上、大蒙古国第一代皇帝・元太祖（1162年5月31日～1227年8月25日）。
<br><br>
デモでは大漠遠行について言及があり、彼は[趙活](/ja/people/characters/player)、[瑞杏](/ja/people/characters/girl1)、[丘真人（丘處機）](/ja/people/characters/special209)を大幕に召し入れて、長生不死の法を知りたいとした。[丘真人](/ja/people/characters/special209)は一言で殺生を止めさせ、大汗は干戈を休んで功徳を養うべきだと勧めた。[瑞杏](/ja/people/characters/girl1)は大汗に天命に順応して、生涯のうちに思い切って行動すべきと勧めた。[趙活](/ja/people/characters/player)は心が痛み、大王ももうすぐ死ぬから戦い尽くせないかもしれないと言った。最終的に[趙活](/ja/people/characters/player)が[瑞杏](/ja/people/characters/girl1)の野心を見たことが成吉思汗に目撃され、指名手配される。
