# {{ $frontmatter.title }}

[丐幫](/ja/people/factions/beggar-gang)旧派の長老、「千金叟」李相公。一子[李富貴](/ja/people/characters/special809)がいる。

<br>

もとは官の功名を持つ身でありながら、江湖の道義のために家財を投げうって[丐幫](/ja/people/factions/beggar-gang)に加わった――ということになっている。実際は、収賄が発覚して朝廷から家財没収の役人を差し向けられ、逃れられぬと悟るや、夜通し財産をばらまいて乞食たちに金品を配り、善名を買って丐幫に潜り込んだのである。盗人猛々しくも、官府に冤罪を着せられたと言い張った。若き日の[王二壯](/ja/people/characters/special401)と[南宮遠](/ja/people/characters/special101)は彼に騙されて善人と思い込み、南宮遠に至ってはその気前の良さに感服さえしていた。

<br>

丐幫入りの後は、恩を着せて[丐幫](/ja/people/factions/beggar-gang)弟子をこき使い阿漕な金儲けをさせ、二十年前に敗れた[旧魔教](/ja/people/factions/elysium-school)の残党を手中に収めた。男は金にならぬと殺し、女は手足の腱を断って手下に慰み者として与えた。生まれた子は金国に売るか、「採生折割」で生まれつきの障害を装わせ、道行く人の同情を引いて金を騙し取った。被害者がみな武林の人間で市井の民ではないため朝廷は取り締まらず、千金叟はその抜け穴を突いて阿漕に荒稼ぎした。

<br>

最期は丐幫大会で[王二壯](/ja/people/characters/special401)の一掌に打たれて死に、遺児[李富貴](/ja/people/characters/special809)は王二壯の義子となった。
