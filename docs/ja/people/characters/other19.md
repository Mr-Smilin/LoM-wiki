# {{ $frontmatter.title }}

[丐幫](/ja/systems/faction)の旧派長老にして「千金叟」李相公。一人の息子[李富貴](/ja/people/characters/other15)がいる。

<br>

元々は身分のある官職にあったが、江湖の義理のために家財を散らして[丐幫](/ja/systems/faction)に入ったという触れ込みである。しかし実際には、縁故者による朝廷からの抄家作戦を逃れるためであり、彼は夜通し金銭を散らして乞食に与え、良い名声を買ってから丐幫に隠れ、盗人が言う捕り手だと嘘をついた。若き日の[王二壯](/ja/people/characters/special3)と[南宮遠](/ja/people/characters/special101)は彼に騙され、良い人だと思い、南宮遠は彼の慷慨さを尊敬していた。

<br>

丐幫入門後、彼は恩義を笠に着て[丐幫](/ja/systems/faction)の弟子を使役して汚い金儲けをした。20年前の戦敗した[極樂教](/ja/systems/faction)の残党を囲い込み、男は価値がないからと殺害。女は手脚の腱を切って、配下の玩具とした。生まれた子供は金国に売るか、それとも「採生折割」で天生の障害者に見立てて、他人の同情心を騙して金銭を詐取した。被害者が武林の人間ばかりで、一般的な民ではなかったため、朝廷は関知しなかった。千金叟はこの隙をついて、悪徳商売に手を染めた。

<br>

最後に、丐幫の大会で[王二壮](/ja/people/characters/special3)に一掌で打ち殺され、遺児[李富貴](/ja/people/characters/other15)は王二壮の養子となった。
