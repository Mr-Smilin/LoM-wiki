# {{ $frontmatter.title }}

史実における金国の駙馬 (12世紀—1221年7月16日)。本名は阿海。
<br><br>
山東宣撫使として『養龍苑』を設立し、宋国の武林人の勧誘・取り込みを専門に行った。
<br><br>
[唐布衣](/ja/people/characters/brother1)の大金行きの際には左副元帥であったが、唐布衣に一刀のもとに首を刎ねられて死んだ。その首は返す刀で暗器代わりに投げられ、追手を一人打ち殺した。

## 備考

- 養龍苑の設定は、龍湘の石灰粉のシナリオに由来する
