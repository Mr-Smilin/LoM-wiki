# {{ $frontmatter.title }}

歴史的事実の金国の駙馬（12世紀〜1221年7月16日）。本名はアハイ。

<br>

山東宣撫使として、『養龍苑』を設立し、宋国の武林人員の勧誘と誘拐を専門とした。

<br>

[唐布衣](/ja/people/characters/brother1)が大金行時には左副元帥を務めていたが、彼に一刀で首を落とされて死亡。その死体を後ろ手で暗器として使い、追兵一人を打ち殺した。

## 備註

- 養龍苑の話の一部は龍湘の石灰粉エピソードから来ている。
