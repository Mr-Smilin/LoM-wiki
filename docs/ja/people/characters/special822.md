# {{ $frontmatter.title }}

前崆峒掌派人の娘で、その身分を頼みに指図して命ずる。激しく快活で非常に剽悍である。[魏菊](/ja/people/characters/wei-ju)の遠い親戚で、容貌に七分の相似がある。[夏侯蘭](/ja/people/characters/xiahou-lan)の生涯で最も親しい友人。
<br><br>
[夏侯蘭](/ja/people/characters/xiahou-lan)が[唐布衣](/ja/people/characters/tang-bu-yi)一行と共に江湖を遍歴している時、[第三香](/ja/people/characters/di-san-xiang)と鴛鴦の盟いを交わす。その後第三香と密かに密会する時、夏侯蘭と[閻羅生](/ja/people/characters/yan-luo-sheng)に見つかり、その混乱の中で、夏侯蘭の為に判官筆を受けて殉職した。
<br><br>
たとえ夏侯蘭が彼女を骨まで恨んでも、それでも恨むことさえ許さず、死ぬ前なお彼女と姉妹になることを望む。その後夏侯蘭は晴天雨天を問わず傘を持ち、たとえ自己欺瞞であっても、呼延菊の亡霊を留めておくことを望む。
