# {{ $frontmatter.title }}

前崆峒掌派の長の娘で、身分を頼りに傲慢に振舞う。非常に剽悍で気が強い。[魏菊](/ja/people/characters/girl7)の遠い親戚で、容貌が七分ほど似ている。[夏侯蘭](/ja/people/characters/girl5)の生涯で最も親しい友人。
<br><br>
[夏侯蘭](/ja/people/characters/girl5)が[唐布衣](/ja/people/characters/brother1)一行と共に江湖を遍歴している間に、[第三香](/ja/people/characters/special820)と鴛鴦の盟約を結ぶ。その後、第三香と密かに逢瀬をしていた時、夏侯蘭と[閻羅生](/ja/people/characters/other2)に見つかり、その混乱の中で、夏侯蘭を庇って判官筆の一撃を身に受け、夭折した。
<br><br>
夏侯蘭がどれほど恨もうとも、呼延菊は恨まれることを許さず、死ぬ前までもなお彼女と姉妹になることを願っていた。その後、夏侯蘭は晴れ日も雨の日も傘を持ち続け、たとえ自己欺瞞であろうとも、呼延菊の面影を心に保ち続けた。
