# {{ $frontmatter.title }}

鐵拳門の『白眉虎』孟瞋。かつて飛天門の[丹霞子](/ja/people/characters/special404)、玄功門の丹青羅刹[閻羅生](/ja/people/characters/other2)、奪魄門の[第三香](/ja/people/characters/special820)、四人で崆峒四秀と並称。鐵砂掌を練得し、掌派の人選を争う望みがあった。

不幸にも[裂骨魔](/ja/people/characters/girl6)の怒りを買い、掌骨全部つぶされた。治っても生涯武功はできない。早市で包子を売っている。
